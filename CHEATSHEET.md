# Шпаргалка: команды Claude Code

## Сценарии (говори как хочешь -- Claude подхватит)

| Что хочешь сделать | Что сказать | Что будет |
|---|---|---|
| **Снять видео** | «делаем видео про [тему]» | Ресёрч конкурентов -> спросит про твой опыт -> бриф -> сценарий -> обложка -> SEO -> shorts -> ревью |
| **Подготовить посты** | «посты на 2 недели» | Спросит про кейсы/задачи -> 6 постов пачкой -> AI-check + quality-check каждого |
| **Создать курс** | «делаем курс по [теме]» | Ресёрч -> спросит про опыт -> структура -> разделы -> гайд -> видео-скрипты |

## Точечные команды (когда нужно что-то одно)

| Что хочешь | Что сказать |
|---|---|
| Найти идеи для видео | «найди темы для видео» или «что снять» |
| Один TG-пост | «напиши пост для телеграма» |
| Обложка для видео | «сделай обложку» |
| Ревью обложки | «оцени обложку» |
| Проверить пост на AI | «проверь пост на AI-маркеры» |
| Сгенерировать картинку | `/generate-image` |

## Как это работает

Во всех сценариях Claude **сам остановится и спросит тебя**:
- Какие кейсы из опыта добавить
- Какую позицию занять
- Одобряешь ли план/бриф/структуру

Ты **не запускаешь скиллы вручную**. Просто описываешь задачу своими словами.
