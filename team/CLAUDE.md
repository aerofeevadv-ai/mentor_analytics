# Team — управление командой

## Scope

Практическая теория управления людьми + применение к команде Mentor Analytics (помощник контента, ops, part-time менторы, фрилансеры-подрядчики).

## Файлы

- `management_20_80.md` — главный файл. Единая теория + шаблоны. Прочитать целиком один раз, использовать как справочник.
- `case_content_assistant.md` — активный кейс: помощница по контенту. Диагностика + план на 30 дней.

## Команда на 2026-04-20

| Сотрудник | Тип | Функция |
|---|---|---|
| Помощница контента | Штат | Публикация в SMM-планере, передача сырья монтажёрам (см. case-файл) |
| Интервьюеры-менторы (Junior/Senior) | Part-time | МОК-интервью, саппорт-встречи |
| Монтажёры | Фриланс | Монтаж YouTube |
| Ops | Планируется, штат | Операционка (см. корневой CLAUDE.md) |

## Принципы папки

1. Минимум теории, максимум применимости
2. Один фреймворк на задачу (не 3 альтернативы)
3. Шаблоны сразу готовы к использованию
4. Учитывает мягкий стиль основателя — через призму Radical Candor (забота + прямота, не жёсткость)
5. Масштабируется на любого сотрудника с адаптацией блоков (см. Блок 9 в management_20_80.md)

## Работа с файлами

При работе с папкой:
- Прежде чем давать совет по конкретному сотруднику — свериться с `management_20_80.md` (особенно Блок 6 про диагностику Will × Skill)
- Разбор конкретного случая — создавать новый `case_*.md` файл рядом
- Шаблоны и фреймворки — только из management_20_80.md, не изобретать новые без нужды
