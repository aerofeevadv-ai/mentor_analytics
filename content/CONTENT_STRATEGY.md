**Вырастить аудиторию для запуска и продажи курсов по аналитике.**

```
         (охват)                      (прогрев)              (деньги)
```

## 5. Площадки

### YouTube — ядро (охват + SEO)

- **Лонги:** 2/мес, нечётные недели
- **Shorts:** 2-3 с каждого лонга (4-6/мес)
- **CTA:** подписка + ссылка на Telegram в описании
- **Контент:** 100% экспертный

### Telegram — центр доверия и продаж

- **Частота:** 3 поста/нед
- **Промт:** `content/telegram/claude_prompt.md`
- **Проверка:** `/tg-quality-check` + `/tg-ai-check` перед апрувом
- **Расписание:**

| День | Нечётная неделя | Чётная неделя |
|---|---|---|
| Вт | Кейс ученика | Образовательный (разбор собеса) |
| Чт | Анонс YouTube-видео | Лучший short + ссылка |
| Вс | Личный инсайт | Лёгкий / ситуативный |

- **Контент:** 90% экспертный, 10% лёгкий

### MAX — зеркало Telegram

1:1 копия всех TG-постов. Помощница дублирует через SMMPlanner или настраиваем бот.

### Instagram — Reels + экспертный пост + личное

- **Reels:** = YouTube Shorts (тот же файл), 2-3/нед
- **Экспертный пост:** вторничный TG-пост (картинка-заглушка + caption), 1/нед — помощница
- **Карусели:** лучшие TG-посты в Canva, 1-2/мес — помощница
- **Личное:** фото + короткий текст (зал, рабочее место, закулисье), 1-2/нед — **Алексей сам**
- **Контент:** 60% экспертный, 40% личный

### TikTok — вирусный охват

- **Контент:** = YouTube Shorts (тот же файл)
- **Частота:** 3-4/нед (на старте — ежедневно 2 недели для прогрева)

### VK — хедж YouTube

- **Видео:** лонги с YouTube (зеркало), 2/мес
- **Клипы:** = Shorts, 2-3/нед
- **Экспертный пост:** вторничный TG-пост (текст), 1/нед — помощница
- **Личное:** то же фото, что и в IG, 1-2/нед — **Алексей сам**
- **Контент:** 70% экспертный, 30% личный

## 6. Переупаковка: один контент → все площадки

### Из лонга YouTube

```
Лонг (оригинал)
 ├─→ 2-3 Shorts → YouTube / Reels / TikTok / VK Клипы
 ├─→ Анонс-пост → Telegram / MAX
 ├─→ Лонг целиком → VK Видео
 └─→ Карусель (иногда) → Instagram
```

### Из TG-поста

```
TG-пост (оригинал)
 ├─→ MAX (1:1 копия, все посты)
 ├─→ IG пост (вторник, картинка + caption)
 └─→ VK пост (вторник, текст)
```

### Личное фото (Алексей сам)

```
Фото + 2-3 предложения
 ├─→ IG пост
 └─→ VK пост
```

### Итого в месяц

| Что | Кол-во | Кто делает |
|---|---|---|
| YouTube лонгов | 2 | Я (запись) + монтажёр |
| Shorts | 4-6 | Монтажёр |
| TG постов | 10-12 | Claude (черновик) → я (апрув) |
| MAX постов | 10-12 | = TG, зеркало |
| IG Reels | 8-12 | = Shorts, помощница |
| IG экспертный пост | 4 | Помощница (вторничный TG-пост) |
| IG карусель | 1-2 | Помощница (Canva) |
| IG/VK личное | 4-8 | Я сам |
| TikTok видео | 12-16 | = Shorts, помощница |
| VK клипов | 8-12 | = Shorts, помощница |
| VK экспертный пост | 4 | Помощница (вторничный TG-пост) |
| VK видео (лонги) | 2 | Помощница (зеркало YouTube) |

**Мне из этого:** записать 2 видео + апрувнуть посты + иногда фото в IG/VK. Всё остальное — команда.

## 9. Воронка под запуск курса

### За 4-6 недель: прогрев

Видео и посты по теме курса. Не продаём — создаём контекст.

### За 2 недели: запуск

| День | Пост | Цель |
|---|---|---|
| -14 | Пост-проблема | Осознание боли |
| -10 | Пост-решение | Экспертность |
| -7 | Анонс курса | Предвкушение |
| -3 | Кейс ученика по теме | Соц. доказательство |
| 0 | Открытие продаж | Конверсия |
| +2 | FAQ | Снятие возражений |
| +5 | Дедлайн / «осталось X мест» | Срочность |

### После запуска: evergreen

Контент по теме курса продолжается, каждый пост — soft-CTA.
