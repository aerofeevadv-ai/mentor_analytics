## Зачем этот документ

Единственный источник правды по контенту. Если чего-то здесь нет — это не делается.

---

## 1. Цель

**Вырастить аудиторию для запуска и продажи курсов по аналитике.**

Менторство в заявках не нуждается — CTA из контента ведёт на Telegram, не на менторство.

```
YouTube / TikTok / Reels / VK  →  подписка на Telegram  →  покупка курса
         (охват)                      (прогрев)              (деньги)
```

## 4. Площадки

### YouTube — ядро (охват + SEO)

- **Лонги:** 2/мес, нечётные недели
- **Shorts:** 2-3 с каждого лонга (4-6/мес)
- **CTA:** подписка + ссылка на Telegram в описании

### Telegram — центр доверия и продаж

- **Частота:** 3 поста/нед
- **Расписание:**

| День | Нечётная неделя | Чётная неделя |
|---|---|---|
| Вт | Кейс ученика | Образовательный контент |
| Чт | Анонс YouTube-видео | Лучшее короткое видео |
| Вс (или Пн) | Личный инсайт | — (или доп. пост) |

### MAX — зеркало Telegram

1:1 копия всех TG-постов. Помощница дублирует или настраиваем бот.

### Instagram — Reels + карусели

- **Reels:** = YouTube Shorts (тот же файл), 2-3/нед
- **Карусели:** переупаковка TG-постов в 5-8 слайдов (Canva), 1-2/нед
- **Stories:** анонсы видео (опционально)

### TikTok — вирусный охват

- **Контент:** = YouTube Shorts (тот же файл)
- **Частота:** 3-4/нед (на старте — ежедневно 2 недели для прогрева)

### VK — хедж YouTube

- **Видео:** лонги с YouTube (зеркало), 2/мес
- **Клипы:** = Shorts, 2-3/нед
- **Посты:** лучшие TG-посты, 1-2/нед

## 5. Переупаковка: один контент → все площадки

### Из лонга YouTube

```
Лонг (оригинал)
 ├─→ 2-3 Shorts → YouTube / Reels / TikTok / VK Клипы
 ├─→ Анонс-пост → Telegram / MAX
 ├─→ Лонг целиком → VK Видео
 └─→ 1-2 карусели (тезисы) → Instagram
```

### Из TG-поста

```
TG-пост (оригинал)
 ├─→ MAX (1:1 копия)
 ├─→ Instagram карусель (Canva шаблон)
 └─→ VK пост (копия или расширенная версия)
```

### Итого в месяц

| Что | Кол-во | Кто делает |
|---|---|---|
| YouTube лонгов | 2 | Я (запись) + монтажёр |
| Shorts | 4-6 | Монтажёр/quso.ai |
| TG постов | 10-12 | Claude (черновик) → я (апрув) |
| IG Reels | 8-12 | = Shorts, помощница заливает |
| IG каруселей | 4-8 | Помощница (Canva) |
| TikTok видео | 12-16 | = Shorts, помощница заливает |
| VK клипов | 8-12 | = Shorts, помощница заливает |
| VK постов/видео | 4-6 | Помощница |
| MAX постов | 10-12 | = TG, зеркало |

**Мне из этого:** записать 2 видео + апрувнуть посты. Всё остальное — команда.

## 8. Воронка под запуск курса

### За 4-6 недель: прогрев

Видео и посты по теме курса. Не продаём — создаём контекст.

### За 2 недели: запуск

| День | Пост | Цель |
|---|---|---|
| -14 | Пост-проблема | Осознание боли |
| -10 | Пост-решение | Экспертность |
| -7 | Анонс курса | Предвкушение |
| -3 | Кейс ученика по теме | Соц. доказательство |
| 0 | Открытие продаж | Конверсия |
| +2 | FAQ | Снятие возражений |
| +5 | Дедлайн / «осталось X мест» | Срочность |

### После запуска: evergreen

Контент по теме курса продолжается, каждый пост — soft-CTA.
