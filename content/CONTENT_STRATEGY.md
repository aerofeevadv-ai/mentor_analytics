## 1. Цель

**Вырастить аудиторию для запуска и продажи курсов по аналитике.**

Менторство в заявках не нуждается — CTA из контента НЕ ведёт на менторство.

### Воронка

```
YouTube / Shorts / TikTok              (охват — видео)
        ↓           ↓          ↓
   Telegram/MAX   IG native   VK native
   (3 поста/нед)  (свой план) (свой план)
        ↓           ↓          ↓
        └─────── Лендинг курса ←──────┘
```

**Принцип:** не гнать всех в Telegram. Аудитория остаётся на своей площадке, прогревается там, покупает через лендинг.

- **YouTube CTA** → подписка на Telegram (самый вовлечённый канал)
- **IG CTA** → подписка в IG + лендинг курса (при запуске)
- **VK CTA** → подписка в VK + лендинг курса (при запуске)
- **TG/MAX CTA** → лендинг курса (при запуске)

### Фаза 2 (после запуска площадок): Email

Лид-магнит (чек-лист, PDF-гайд) → сбор email → дублирование запусков курса в рассылку. Email — единственная база, которую не отберёт ни одна платформа. Не первый приоритет, но стратегический хедж.

## 5. Площадки

### YouTube — ядро (охват + SEO)

- **Лонги:** 2/мес, нечётные недели
- **Shorts:** 2-3 с каждого лонга (4-6/мес)
- **CTA:** подписка + ссылка на Telegram в описании
- **Контент:** 100% экспертный

### Telegram — самый глубокий канал

- **Частота:** 3 поста/нед
- **Промт:** `content/telegram/claude_prompt.md`
- **Проверка:** `/tg-quality-check` + `/tg-ai-check` перед апрувом
- **CTA:** подписка на канал, при запуске курса → лендинг
- **Расписание:**

| День | Нечётная неделя | Чётная неделя |
|---|---|---|
| Вт | Кейс ученика | Образовательный (разбор собеса) |
| Чт | Анонс YouTube-видео | Лучший short + ссылка |
| Вс | Личный инсайт | Лёгкий / ситуативный |

- **Контент:** 90% экспертный, 10% лёгкий

### MAX — зеркало Telegram

1:1 копия всех TG-постов. Помощница дублирует через SMMPlanner или настраиваем бот.

### Instagram — свой контент-план (Reels + посты + личное)

- **Reels:** = YouTube Shorts (тот же файл), 2-3/нед
- **Экспертный пост:** вторничный TG-пост (картинка-заглушка + caption), 1/нед — помощница
- **Карусели:** лучшие TG-посты в Canva, 1-2/мес — помощница
- **Личное:** фото + короткий текст (зал, рабочее место, закулисье), 1-2/нед — **Алексей сам**
- **Контент:** 60% экспертный, 40% личный
- **CTA:** подписка в IG, при запуске курса → лендинг. НЕ гнать в Telegram

### TikTok — вирусный охват

- **Контент:** = YouTube Shorts (тот же файл)
- **Частота:** 3-4/нед (на старте — ежедневно 2 недели для прогрева)

### VK — свой контент-план (видео + посты + личное)

- **Видео:** лонги с YouTube (зеркало), 2/мес
- **Клипы:** = Shorts, 2-3/нед
- **Экспертный пост:** вторничный TG-пост (текст), 1/нед — помощница
- **Личное:** то же фото, что и в IG, 1-2/нед — **Алексей сам**
- **Контент:** 70% экспертный, 30% личный
- **CTA:** подписка в VK, при запуске курса → лендинг. НЕ гнать в Telegram

## 8. Как выбирать темы для YouTube

Не придумывать из головы. Два источника тем:

### Источник 1: Конкуренты (что уже работает)

Раз в 2 недели запускать `/youtube-ideas` → скрипт анализирует 30+ каналов (RU + EN), считает virality score.

```
/youtube-ideas              ← discover: топ видео конкурентов за 3-6 мес
/youtube-ideas "тема"       ← topic: проверить конкретную тему
```

Брать видео с score 6+ и адаптировать под свой угол.

### Источник 2: Курсы (контент = превью курса)

Каждое видео может быть демо одного модуля/урока из курса:
- «5 метрик, которые спрашивают на каждом собесе» → превью модуля 1 курса продуктовой аналитики
- «Как решать продуктовые кейсы — метод КМДГП» → превью модуля 2
- «A/B тесты: что спрашивают на собесе» → превью курса A/B тестов

CTA в конце: «это 1 из N тем в моём курсе, ссылка в описании».

### Процесс выбора темы

```
1. Запустить /youtube-ideas (раз в 2 нед)
2. Из топ-10 отфильтровать: релевантно аудитории? Есть связь с курсом?
3. Проверить: не снимал ли уже (content/youtube/ideas.md)
4. Выбрать 1 тему → запустить /youtube-scripts
```

### Баланс

- ~50% видео — конкурент-driven (проверенный спрос, охват)
- ~50% видео — курс-driven (прямая связь с продуктом, продажи)

## 9. Продажа курсов через контент

### Контекст

Курсы — это миникурсы (текст + видео в Notion/Teamly) по 6-25К ₽, не программы за 100К. Это значит:
- Импульсная покупка — длинный прогрев не нужен
- Контент = демо курса (показывай куски → «остальное тут»)
- Evergreen важнее запусков — курс всегда доступен
- Подробнее о курсах и ценах → `courses/PRODUCT_STRATEGY.md`

### Первый запуск (один раз для каждого курса)

Лёгкий запуск на 3-5 дней. Цель — осведомлённость, не искусственная срочность.

| День | Пост (все площадки) | Цель |
|---|---|---|
| -3 | Полезный пост по теме курса | Контекст |
| 0 | Анонс: что за курс, для кого, ссылка на лендинг | Открытие продаж |
| +2 | Кейс / отзыв бета-тестера | Соц. доказательство |
| +4 | Повтор: для тех кто пропустил | Охват |

**Запуск — на всех площадках одновременно:** TG, MAX, IG, VK. Все CTA ведут на один лендинг.

### После запуска: evergreen (основной режим)

Курс всегда доступен. Ссылка стоит в:
- Описании YouTube-канала и каждого релевантного видео
- Bio IG и VK
- Закреплённом сообщении TG/MAX

**Контент = демо курса.** Регулярно показывать кусок из курса как обычный пост/видео. Не продавать, а демонстрировать:
- «Вот как разбирать продуктовые кейсы — метод КМДГП. Это 1 из 10 типов кейсов в моём курсе.»
- «SQL-задача с собеса в Яндексе — разбор. Ещё 15 задач с разборами в курсе.»

**Периодические акции** (раз в 1-2 мес, не чаще):
- Скидка 20% на неделю
- Бандл со скидкой (2 курса дешевле)
- Не дедлайны и «осталось 3 места» — это для базового тарифа неуместно

**Тариф с консультацией — отдельно.** Лимит 4 места/мес — реальное ограничение. Здесь скарсити уместна.

### Неделя 1 (перезапуск IG и VK)

Текущие подписчики знают Алексея лично, но не знают про аналитику и менторство. Нужна серия вводных постов — «вот чем я теперь занимаюсь».

**3-4 вводных поста (IG + VK), публиковать через день:**

1. **Кто я сейчас** — личное фото + текст: чем занимаюсь, что за «Аналитика без воды», почему ушёл из найма в менторство. Коротко, от первого лица.
2. **Что такое аналитик данных** — простым языком для тех, кто не в теме. Что делает, сколько зарабатывает, почему это востребовано.
3. **Что такое менторство** — как устроено, чем отличается от курсов, кому подходит. Можно с кейсом ученика.
4. **Что будет на этой странице** — «буду делиться экспертным контентом + немного личного. Подписывайтесь, если интересна аналитика или карьера в IT.»
