## 1. Цель

**Вырастить аудиторию для запуска и продажи курсов по аналитике.**

Менторство в заявках не нуждается — CTA из контента НЕ ведёт на менторство.

### Воронка

```
YouTube / Shorts / TikTok              (охват — видео)
        ↓           ↓          ↓
   Telegram/MAX   IG native   VK native
   (3 поста/нед)  (свой план) (свой план)
        ↓           ↓          ↓
        └─────── Лендинг курса ←──────┘
```

**Принцип:** не гнать всех в Telegram. Аудитория остаётся на своей площадке, прогревается там, покупает через лендинг.

- **YouTube CTA** → подписка на Telegram (самый вовлечённый канал)
- **IG CTA** → подписка в IG + лендинг курса (при запуске)
- **VK CTA** → подписка в VK + лендинг курса (при запуске)
- **TG/MAX CTA** → лендинг курса (при запуске)

### Фаза 2 (после запуска площадок): Автоворонки + Email

- **VK → Senler-автоворонка** (рассылки ВК, открываемость до 80%): подписка на группу → лид-магнит → прогрев → оффер курса. Подробнее → раздел 12, п.14.
- **Email** (стратегический хедж): лид-магнит → сбор email → дублирование запусков курса. Email — единственная база, которую не отберёт ни одна платформа.

## 5. Площадки

### YouTube — ядро (охват + SEO)

- **Лонги:** 2/мес, нечётные недели
- **Shorts:** 2-3 с каждого лонга (4-6/мес)
- **CTA:** подписка + ссылка на Telegram в описании
- **Контент:** 100% экспертный

### Telegram — самый глубокий канал

- **Частота:** 3 поста/нед
- **Промт:** `content/telegram/claude_prompt.md`
- **Проверка:** `/tg-quality-check` + `/tg-ai-check` перед апрувом
- **CTA:** подписка на канал, при запуске курса → лендинг
- **Расписание:**

| День | Нечётная неделя | Чётная неделя |
|---|---|---|
| Вт | Кейс ученика | Образовательный (разбор собеса) |
| Чт | Анонс YouTube-видео | Лучший short + ссылка |
| Вс | Личный инсайт | Лёгкий / ситуативный |

- **Контент:** 90% экспертный, 10% лёгкий

### MAX — зеркало Telegram

1:1 копия всех TG-постов. Помощница дублирует через SMMPlanner или настраиваем бот.

### Instagram — свой контент-план (Reels + посты + личное)

- **Reels:** = YouTube Shorts (тот же файл), 2-3/нед
- **Экспертный пост:** вторничный TG-пост (картинка-заглушка + caption), 1/нед — помощница
- **Карусели:** лучшие TG-посты в Canva, 1-2/мес — помощница
- **Личное:** фото + короткий текст (зал, рабочее место, закулисье), 1-2/нед — **Алексей сам**
- **Контент:** 60% экспертный, 40% личный
- **CTA:** подписка в IG, при запуске курса → лендинг. НЕ гнать в Telegram

### TikTok — вирусный охват

- **Контент:** = YouTube Shorts (тот же файл)
- **Частота:** 3-4/нед (на старте — ежедневно 2 недели для прогрева)

### VK — свой контент-план (видео + посты + личное)

**Три точки присутствия:**
- **Группа** («Аналитика без воды») — основная: видео, клипы, экспертные посты. Единая точка подписки, CTA везде ведёт в группу.
- **Канал сообщества** — хронологическая лента (аналог TG-канала). Охват доставки ~50% подписчиков — выше алгоритмической ленты. Быстрые заметки, ссылки, анонсы. **Не дублировать стену.**
- **Личная страница** — лицо: репосты из группы + личные фото. Экспертный контент НЕ публикуется на личной странице напрямую.

**Схема публикации:**

```
Видео (лонги + клипы)          → группа (лонги одновременно с YouTube или раньше)
Все TG-посты (3/нед)           → группа (текст) → репост на личную страницу
Быстрые заметки / ссылки       → канал сообщества (ежедневно, не дублировать стену)
Личное фото + текст            → личная страница (не в группу)
```

- **Видео:** лонги с YouTube, 2/мес — группа. **Публиковать одновременно с YouTube или раньше** (алгоритм «Немезида» считает первоисточником того, кто опубликовал раньше внутри VK)
- **Клипы:** 4-5 с каждого лонга (8-10/мес, ~3-5/нед) — группа. **Привязывать к лонгу** через механику VK (кнопка «Перейти в полную версию», до 20 клипов на видео)
- **Посты в группе:** все TG-посты (Вт/Чт/Вс), 3/нед — помощница через SMMPlanner. Вторничный пост репостить на личную страницу
- **Канал сообщества:** ежедневные быстрые заметки — ссылка на новое видео, мысль дня, опрос. Помощница или Алексей
- **Личное:** то же фото, что и в IG, 1-2/нед — **Алексей сам** на личную страницу
- **Контент:** 70% экспертный, 30% личный
- **CTA:** подписка на группу VK, при запуске курса → лендинг. НЕ гнать в Telegram

**Оформление сообщества (чек-лист при запуске):**
- Живая обложка (1920×1080, до 5 слайдов): лого + позиционирование → УТП с цифрами → ближайший курс → отзывы
- Меню (до 7 пунктов): Каталог курсов → Бесплатный гайд → Отзывы → FAQ → Наши менторы → Связаться
- Кнопка действия «Записаться» → лид-форма VK
- Виджет приветствия (разный текст для новых / подписчиков)
- Закреплённый пост: навигатор по сообществу + CTA на лид-магнит. Обновлять при каждом запуске курса
- Курсы в разделе «Товары» — карточки с ценой и ссылкой на лендинг

**Алгоритмические заметки (VK Видео, 2025-2026):**
- Клипы: первые 3 секунды решают судьбу, 97% просмотров — из рекомендаций. Титры обязательны (50%+ смотрят без звука)
- Лонги: удержание 30%+ — главный фактор, затем CTR обложки, затем активность в первые 1-3 часа
- «Прометей»: нейросеть награждает уникальный контент меткой огня (7 дней кратного роста охватов). 70% получивших — каналы <10К подписчиков
- Discovery (с ноября 2025): алгоритм выводит пользователей из «рекомендательного пузыря» — шанс для новых авторов
- Посты с >40% AI-текста без уникальных фото/видео режутся в охватах. Репосты чужого контента даже 1/нед снижают охваты на 15-20%
- Охваты приходят волнами (2-й, 3-й, 5-й день) — оценивать эффективность поста не раньше чем через 5 дней
- Карусели — «открытие 2025 года», в B2B-нишах обгоняют даже вертикальное видео
- При 1 000+ подписчиков → подать на **VK Гранты** (до 5 млн ₽, образовательный контент в приоритете, доступно самозанятым)
- При 5 000+ подписчиков → подключить **Revenue Share** (50-80% от рекламных врезок)

## 5.1. Instagram в VK-контенте — юридические правила

> Meta признана экстремистской организацией в РФ. С 2025-2026 правоприменение резко ужесточилось: реальные аресты за логотип IG на сайте (15 суток, февраль 2026), штрафы блогерам-ИП, законопроект о 10-кратном увеличении штрафов.

**Абсолютный запрет (для всего VK-контента):**
- Никогда не показывать логотип Instagram / Facebook / Meta в видео, клипах, обложках, постах
- Никогда не давать прямую ссылку на Instagram-аккаунт (@mentor.analytics, instagram.com/...)
- Никогда не призывать «подписывайтесь в Instagram» в VK-контенте

**Безопасные альтернативы:**
- CTA «Все контакты — в описании» → ссылка на link-in-bio (mentoranalytics.ru/links)
- Эвфемизмы: «запрещёнка», «запрещённая соцсеть», «фотосоцсеть» — если нужно упомянуть
- Если упоминаем слово «Instagram» текстом — обязательна маркировка: *принадлежит Meta Platforms Inc., признанной экстремистской организацией, деятельность которой запрещена на территории РФ*

**Риски для ИП:**
- Логотип IG в видео → штраф 1 000–4 000 ₽ или арест до 15 суток (ст. 20.3 КоАП)
- Прямая ссылка без маркировки → штраф до 5 000 ₽ (ст. 13.15 КоАП)
- Призыв подписаться → потенциально ст. 282.2 УК (вовлечение в деятельность экстремистской организации)
- Конкурентный донос — основной механизм запуска дел

**Чек-лист для помощницы (при публикации на VK):**
- [ ] В видео нет логотипа IG / Meta / Facebook
- [ ] В описании нет прямой ссылки на Instagram
- [ ] Если IG упомянут текстом — есть маркировка со звёздочкой
- [ ] CTA ведёт на группу VK или link-in-bio, не на IG

**Link-in-bio (mentoranalytics.ru/links):**
- Кнопка Instagram — нейтральный текст (например, «Фото и закулисье»), **без логотипа IG**
- Промежуточная страница создаёт правовую дистанцию между VK-контентом и IG

## 6. Переупаковка: один контент → все площадки

### Из лонга YouTube

```
Лонг (оригинал)
 ├─→ 4-5 Shorts (2 версии каждого: с плашкой + чистая)
 │    ├─→ С плашкой (YT+TG) → YouTube Shorts
 │    └─→ Чистая (без плашки) → IG Reels / VK Клипы / TikTok
 ├─→ VK Клипы привязать к лонгу (кнопка «Перейти в полную версию»)
 ├─→ Анонс-пост → Telegram / MAX
 ├─→ Лонг целиком → VK Видео (одновременно с YouTube)
 └─→ Карусель (иногда) → Instagram
```

### Из TG-поста

```
TG-пост (оригинал)
 ├─→ MAX (1:1 копия, все посты)
 ├─→ VK группа (все посты, текст) → вторничный репост на личную страницу
 └─→ IG пост (вторник, картинка + caption)
```

### Личное фото (Алексей сам)

```
Фото + 2-3 предложения
 ├─→ IG пост
 └─→ VK пост
```

### Оформление коротких видео по площадкам

На YouTube shorts связаны с лонгами (встроенная механика). На остальных площадках shorts — **самостоятельные единицы контента**.

#### Две версии каждого short

Плашка-картинка в конце шортса почти не влияет на подписки (зрители свайпают до неё, кнопка «подписаться» уже встроена в интерфейс всех платформ). Единственное, что работает — **голосовой CTA в середине видео**. Визуальная плашка оставлена только на YouTube (единственный способ напомнить про Telegram, т.к. end screens не работают в Shorts).

| Версия | Плашка (последние ~3 сек) | Куда заливать |
|---|---|---|
| **С плашкой** | YouTube + Telegram | YouTube Shorts |
| **Чистая** | Без плашки | IG Reels, VK Клипы, TikTok |

**Почему VK — чистая версия:**
- VK Клипы не продвигают в рекомендации ролики с логотипами конкурентов (YouTube, TikTok, IG)
- 97% просмотров клипов — из рекомендаций → чужой логотип = потеря охвата
- CTA — голосом в видео + текст в описании клипа

**Для монтажёра:** рендерить 2 версии каждого short. Разница — только последние ~3 сек (плашка YT+TG / чисто). Плашку сделать как шаблон, подставлять в конец.

#### Переходный план (старые наборы)

| Набор | YouTube | VK Клипы | IG Reels | TikTok |
|---|---|---|---|---|
| 1-2 (проекты удалены) | Уже залиты | Обрезать плашку → чистое видео | VK/IG-версия как есть | Обрезать плашку |
| 3 (проект есть) | Переделать: YT+TG | Переделать: чистая | Чистая | Чистая |
| 4+ (новые) | С плашкой YT+TG | Чистая | Чистая | Чистая |

Обрезку старых видео делает помощница (CapCut, 5 мин на видео). Обрезанный голос в последних секундах — допустимо, это разовая мера для 6-8 видео.

#### Описание (caption) — шаблон для IG / VK / TikTok

```
[Хук — 1 предложение, повторяющее суть видео]

[1-2 предложения контекста / дополнительная ценность, которой нет в видео]

[CTA площадки]
```

**CTA по площадкам:**
- **IG:** «Подписывайся, чтобы не пропустить разборы собесов» или «Полное видео — ссылка в шапке профиля»
- **VK:** «Подписывайся на группу — разбираю аналитику без воды» или «Полный разбор — в видео на канале (ссылка в описании группы)»
- **TikTok:** «Подписывайся — разбираю аналитику каждый день»

**Связь с лонгом — мягкая, max 1 из 3 роликов.** Не писать «полное видео на YouTube» в каждом — платформы не любят уводить аудиторию, и люди не переходят. Использовать только если short интригует и обрывается на полуслове: «Полный разбор — ссылка в шапке профиля».

**Описание ≠ копия YouTube.** YouTube-описание оптимизировано под SEO. Для IG/VK/TikTok — разговорный caption. Не копировать.

**Хештеги:**
- **IG:** 5-10 (микс тематических + охватных): `#аналитикданных #собеседование #карьеравIT #datascience #аналитика`
- **VK:** 3-5
- **TikTok:** 3-5

**Чек-лист для помощницы (на каждый short):**

## 8. Как выбирать темы для YouTube

Не придумывать из головы. Два источника тем:

### Источник 1: Конкуренты (что уже работает)

Раз в 2 недели запускать `/youtube-ideas` → скрипт анализирует 30+ каналов (RU + EN), считает virality score.

```
/youtube-ideas              ← discover: топ видео конкурентов за 3-6 мес
/youtube-ideas "тема"       ← topic: проверить конкретную тему
```

Брать видео с score 6+ и адаптировать под свой угол.

### Источник 2: Курсы (контент = превью курса)

Каждое видео может быть демо одного модуля/урока из курса:
- «5 метрик, которые спрашивают на каждом собесе» → превью модуля 1 курса продуктовой аналитики
- «Как решать продуктовые кейсы — метод КМДГП» → превью модуля 2
- «A/B тесты: что спрашивают на собесе» → превью курса A/B тестов

CTA в конце: «это 1 из N тем в моём курсе, ссылка в описании».

### Процесс выбора темы

```
1. Запустить /youtube-ideas (раз в 2 нед)
2. Из топ-10 отфильтровать: релевантно аудитории? Есть связь с курсом?
3. Проверить: не снимал ли уже (content/youtube/ideas.md)
4. Выбрать 1 тему → запустить /youtube-scripts
```

### Баланс

- ~50% видео — конкурент-driven (проверенный спрос, охват)
- ~50% видео — курс-driven (прямая связь с продуктом, продажи)

## 9. Продажа курсов через контент

### Контекст

Курсы — это миникурсы (текст + видео в Notion/Teamly) по 6-25К ₽, не программы за 100К. Это значит:
- Импульсная покупка — длинный прогрев не нужен
- Контент = демо курса (показывай куски → «остальное тут»)
- Evergreen важнее запусков — курс всегда доступен
- Подробнее о курсах и ценах → `courses/PRODUCT_STRATEGY.md`

### Первый запуск (один раз для каждого курса)

Лёгкий запуск на 3-5 дней. Цель — осведомлённость, не искусственная срочность.

| День | Пост (все площадки) | Цель |
|---|---|---|
| -3 | Полезный пост по теме курса | Контекст |
| 0 | Анонс: что за курс, для кого, ссылка на лендинг | Открытие продаж |
| +2 | Кейс / отзыв бета-тестера | Соц. доказательство |
| +4 | Повтор: для тех кто пропустил | Охват |

**Запуск — на всех площадках одновременно:** TG, MAX, IG, VK. Все CTA ведут на один лендинг.

### После запуска: evergreen (основной режим)

Курс всегда доступен. Ссылка стоит в:
- Описании YouTube-канала и каждого релевантного видео
- Bio IG и VK
- Закреплённом сообщении TG/MAX

**Контент = демо курса.** Регулярно показывать кусок из курса как обычный пост/видео. Не продавать, а демонстрировать:
- «Вот как разбирать продуктовые кейсы — метод КМДГП. Это 1 из 10 типов кейсов в моём курсе.»
- «SQL-задача с собеса в Яндексе — разбор. Ещё 15 задач с разборами в курсе.»

**Периодические акции** (раз в 1-2 мес, не чаще):
- Скидка 20% на неделю
- Бандл со скидкой (2 курса дешевле)
- Не дедлайны и «осталось 3 места» — это для базового тарифа неуместно

**Тариф с консультацией — отдельно.** Лимит 4 места/мес — реальное ограничение. Здесь скарсити уместна.
