## ТЗ #1: «Как выбрать IT профессию» (uE1B74LNbuI)

**Текущий CTR:** 3.2% | **Показы:** 34 309 | **Приоритет:** ВЫСШИЙ

### Проблема текущей обложки
- Белая доска с 7 мелкими логотипами (JS, C++, DevOps, UI/UX, Backend, AI/ML) — не считываются на мобильном
- Текст «ПОЛНЫЙ ГАЙД ПО IT ПРОФЕССИЯМ» — длинный, скучный
- Ассоциация с лекцией/занудством

### Вариант A — Split (рекомендуется)

**Референс:** `outputs/images/thumb_it_v1_split.png`
**Фото:** `My foto/0H1A4075.jpg` (ладонь стоп / озадаченный)

**Композиция:**
- Левая половина: озадаченное/растерянное лицо, тёмный холодный фон (синий/серый)
- Правая половина: уверенное/улыбающееся лицо, тёплый яркий фон (золотистый)
- Чёткая вертикальная линия-разделитель по центру

**Текст (добавить вручную):**
- Левая сторона: «КУДА?» — белый жирный шрифт, крупно по центру
- Правая сторона: зелёная галочка или стрелка вверх
- Нижняя полоса (опционально): ничего — заголовок видео даст контекст

**Почему сработает:** Split — проверенный формат на канале (5.5-5.6% CTR). Эмоциональный контраст привлекает внимание.

## ТЗ #2: «4 задачи в рекомендациях WB» (zV3ntjczD38)

**Текущий CTR:** 2.8% | **Показы:** 3 516 | **Приоритет:** средний

### Проблема текущей обложки
- Абстрактные иконки (%, картинка, куб, карточки) — не несут смысла
- «МИЛЛИОНЫ ₽/ДЕНЬ» — неправдоподобный кликбейт для аналитика
- Нет конкретного обещания зрителю

### Новая обложка — формула «Бренд + инсайд»

**Эталон:** обложка «WB изнутри» (7.6% CTR) — использовать тот же визуальный язык

**Фото:** `My foto/0H1A4136.jpg` (руки скрещены, серьёзный) или из съёмки WB-видео

**Композиция:**
- Левая 50%: мужчина, серьёзное/задумчивое лицо
- Правая 50%: логотип WB крупно + скриншот реальных рекомендаций WB (карточки товаров)
- Фон: тёмный с фиолетовым/синим градиентом (фирменные цвета WB)

**Текст (добавить вручную):**
- Крупно: «4 КЕЙСА» — белый жирный
- Под ним: «WB РЕКОМЕНДАЦИИ» — меньше, голубой/фиолетовый
- Без «МИЛЛИОНЫ ₽/ДЕНЬ» — убрать кликбейт

**Почему сработает:** тот же стиль, что у лучшей обложки канала. Бренд WB = узнаваемость. Конкретика «4 кейса» вместо абстракции.

## ТЗ #3: «ТОП-10 вопросов на IT-собеседовании» (Gh-TLFGwJ1A)

**Текущий CTR:** 4.2% | **Показы:** 12 676 | **Приоритет:** средний

### Проблема текущей обложки
- «ОТВЕТЬ ТАК — ПОЛУЧИ ОФФЕР» — текст сверху жёлтым, теряется
- Ноутбук в руках с нечитаемым текстом — бессмысленный элемент
- Нет визуальной интриги / конкретного вопроса

### Новая обложка — формула «Провокация»

**Фото:** `My foto/0H1A4047.jpg` (удивлённый, руки у лица) или `My foto/0H1A4069.jpg` (задумчивый)

**Композиция:**
- Левая 40%: мужчина, стрессовая/напряжённая эмоция (как перед собеседованием)
- Правая 60%: крупная плашка/баббл с конкретным вопросом собеседования
- Фон: тёмный, офисный

**Текст (добавить вручную):**
- Крупная плашка справа: «РАССКАЖИТЕ О СЕБЕ» или «ВАШ ГЛАВНЫЙ ФАКАП?» — один конкретный вопрос
- Мелко внизу или в углу: «ТОП-10» в кружке/бейдже

**Альтернативный вариант:**
- Split: слева — стресс/паника (красный фон), справа — уверенная улыбка + «ОФФЕР» (зелёный фон)
- По центру: «10 ВОПРОСОВ»

**Почему сработает:** конкретный вопрос создаёт любопытство («а как бы я ответил?»). Эмоция стресса знакома каждому, кто готовится к собеседованию.

## ТЗ #4: Заголовок «Где в IT РЕАЛЬНО платят» (p37_JzZdXc8)

**Текущий CTR:** 4.7% | **Обложку НЕ менять** (яркая, с деньгами, работает)

### Проблема
Текущий заголовок: «Где в IT РЕАЛЬНО платят: ТОП 5 профессий с ВЫСОКИМИ зарплатами» — 67 символов, обрезается на мобильных. Нет конкретных цифр.

### Новый заголовок
```
Где в IT платят 300 000+ ₽: ТОП-5 профессий (54 символа)
```

**Почему лучше:**
- 54 символа — не обрезается
- Конкретная цифра 300 000 ₽ — привлекает внимание
- Убраны слова-паразиты «РЕАЛЬНО», «ВЫСОКИМИ»

## ТЗ #5: Заголовок + обложка «ТОП-6 инструментов» (pZLij7bl8Tc)

**Текущий CTR:** 4.8%

### Проблема
- Заголовок: «Вот что на самом деле используют аналитики: ТОП-6 нужных инструментов» — 72 символа, обрезается. Начинается с «Вот что» — не цепляет.
- Обложка: «ТОП-6 НУЖНЫХ ИНСТРУМЕНТОВ ДЛЯ АНАЛИТИКА» — дублирует заголовок

### Новый заголовок
```
ТОП-6 инструментов аналитика в 2026 (36 символов)
```

### Доработка обложки
- Убрать текст «ТОП-6 НУЖНЫХ ИНСТРУМЕНТОВ ДЛЯ АНАЛИТИКА» с обложки
- Оставить только число «6» крупно + логотипы инструментов
- Обложка показывает ЧТО (логотипы), заголовок даёт КОНТЕКСТ (для кого, когда)
