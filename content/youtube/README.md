### `/youtube-comments` — Комментарии

**Файл:** `.claude/skills/youtube-comments/SKILL.md`

Что внутри:
- Классификация комментариев (удалить / игнорировать / ответить)
- Стиль ответов (короткие, разговорные, со своим мнением)
- Шаблоны типовых ответов

Workflow:
1. Claude парсит комменты без ответа → заливает в Google Sheets с черновиками
2. Алексей копирует ответы и **публикует вручную** на YouTube
3. Если Алексей пишет правку в колонку «Правка» → Claude переписывает черновик

> **Claude НЕ публикует ответы на YouTube** — только готовит черновики в Google Sheets.

Когда использовать:
- Разобрать новые комментарии → `/youtube-comments`
- Переписать черновики по правкам Алексея

## Скрипты — как запускать

### Авторизация (первый раз или обновление токена)

```bash
python3 scripts/youtube_auth.py
```
Откроет браузер → авторизация Google → сохранит токен в `.secrets/youtube_token.json`.

> Токен в режиме Testing — refresh_token может истечь через 7 дней. При ошибке авторизации — перезапустить.

### Обновление тегов

```bash
python3 scripts/youtube_update_tags.py            # dry-run
python3 scripts/youtube_update_tags.py --apply     # применить
```
Теги прописаны в словаре `TAGS_MAP` внутри скрипта. Обновляет только видео без тегов.

### Обновление заголовков

```bash
python3 scripts/youtube_update_titles.py           # dry-run
python3 scripts/youtube_update_titles.py --apply    # применить
```
Новые заголовки в словаре `TITLES` внутри скрипта. Для новых правок — добавить `video_id: "новый заголовок"` в словарь.

### Генерация обложек

```bash
# Без фото-референса
python3 scripts/generate_image.py "описание обложки"

# С фото-референсом
python3 scripts/generate_image.py --photo "content/youtube/My foto/0H1A4075.jpg" "описание"

# С именем файла
python3 scripts/generate_image.py --photo photo.jpg --name thumb_name "описание"
```
Требует переменную `POLZA_AI_KEY` в `~/.zshrc`. Результат в `outputs/images/`.

> Gemini Flash Image часто генерирует вертикальные изображения, даже при запросе 16:9. Рекомендуется кропать в Canva/Figma.

### Комментарии → Google Sheets

```bash
python3 scripts/youtube_comments_to_sheets.py              # все без ответа
python3 scripts/youtube_comments_to_sheets.py --days 30     # только за 30 дней
```
Парсит комменты без ответа автора (проверяет оба ID: канал + личный аккаунт).
Создаёт/обновляет Google Sheets таблицу. ID таблицы в `.secrets/comments_sheet_id.txt`.

> Скрипт `youtube_post_replies.py` — backup для автопостинга, в текущем workflow **не используется**. Алексей публикует ответы вручную.

### Паттерны CTR (краткая выжимка)

**Повышает (+1-3%):** конкретные цифры, split до/после, сильная эмоция, узнаваемый бренд, провокация, 2-3 крупных элемента.

**Убивает (-1-3%):** абстрактные иконки, 6+ мелких элементов, длинный текст, нечитаемый текст на экране, кликбейт без конкретики.

## API-квоты

YouTube Data API v3: **10 000 единиц/день** (сбрасывается в полночь Pacific Time).

| Операция | Стоимость |
|---|---|
| videos.list | 1 |
| videos.update (теги, заголовки) | 50 |
| commentThreads.list | 1 |
| comments.insert (ответ) | 50 |

При массовых операциях (29 тегов = 1 450 единиц) квота может закончиться. Скрипты делают `sleep(0.5)` между запросами.
