# ТЗ на редизайн обложек: 3 видео x 2 варианта

**Дата:** 2026-03-16
**Цель:** поднять CTR с текущих 2.8–4.2% до 5%+ через A/B тест (Test & Compare)
**Формат:** 1920 x 1080 px, JPG, до 2 МБ
**Обязательно:** читаемость на 320x180 px (мобильный тест)

---

## Общие правила для всех обложек

- **Максимум 2-3 слова** текста (доказано: оптимум для CTR)
- **Контраст > 4.5:1** (иначе -31% CTR на мобильных)
- **Без тёмных фонов** (Dark Mode YouTube = потеря до 60% CTR)
- **Лицо Алексея** с настоящей эмоцией (не нейтральное, не «YouTube face»)
- **Один фокальный элемент** (A/B тест Malewicz: упрощение = 2x CTR)
- Цвета: **синий + оранжевый** (tech/edu: +25%) или **жёлтый + чёрный** (8.2% avg)
- Текст на обложке **НЕ дублирует** заголовок видео

| **Правая часть (55%)** | Яркий фон: оранжевый (#E67E22) → жёлтый. Алексей с уверенной улыбкой, показывает палец вверх. Один крупный элемент — логотип аналитики (график/таблица). Текст: нет (заголовок видео даёт контекст) |

### Как оценивать

YouTube Test & Compare оптимизирует по **watch time**, не CTR. Но мы также отслеживаем:
- CTR до и после (YouTube Studio → Аналитика → Охват)
- Views velocity (просмотры/день) до и после

### Что делать с результатами

- Если оба варианта побеждают текущую — оставляем победителя
- Если ни один не побеждает — анализируем, создаём вариант C на основе инсайтов
- Паттерн победителей фиксируем в SKILL.md для будущих обложек
