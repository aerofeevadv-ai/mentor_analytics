# Аудит резюме менти

## Как провести аудит

1. Скопируй текст резюме менти в файл `audit/mentee_name.txt`
2. Попроси Claude Code: "Проведи аудит резюме из файла audit/mentee_name.txt"
3. Получишь структурированный разбор с скорингом и конкретными правками

## Формат входных данных

Лучший вариант — скопировать текст резюме с hh.ru как есть.
Можно также PDF или скриншот — Claude Code прочитает.

## Что получишь на выходе

- Скоринг по 10 критериям (до 100 баллов)
- 5 конкретных правок с переформулировками
- Список недостающих ключевых слов
- Оценка уровня (junior / middle / senior)
- 3 стратегических рекомендации
