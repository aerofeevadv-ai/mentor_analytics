## 5. Новая страница: /thanks-product-metrics

**Назначение:** страница после успешной оплаты курса.

**Макет:** простая, белый фон, контент по центру. Минимальная шапка — только логотип «Mentor Analytics.», без меню.

### Доставка доступа — два канала

У каждого тарифа своя ссылка на материалы (разные папки в Teamly). Доступ доставляется двумя способами одновременно:

**Канал 1: Thank-you страница с параметром тарифа**

Redirect URL из платёжки: `mentoranalytics.ru/thanks-product-metrics?tariff=base|pro|consult`

На странице 3 блока со ссылками на материалы (по одному на тариф). Все скрыты по умолчанию. JS (~10 строк) читает параметр `tariff` из URL и показывает нужный блок.

```
✅ Оплата прошла успешно!

Спасибо за покупку курса «Продуктовые метрики и кейсы».

Доступ к курсу:
[Кнопка: Перейти к курсу →]  ← ссылка зависит от параметра tariff

---

Что дальше:
1. Начни с модуля 1 — там введение и план прохождения
2. Рекомендую проходить по 1-2 урока в день
3. Вопросы по курсу: mentoranalytics@yandex.ru
4. Подпишись на канал — t.me/analytic_offers
```

Примерная логика JS:
```js
const tariff = new URLSearchParams(window.location.search).get('tariff');
document.querySelectorAll('[data-tariff]').forEach(el => {
  el.style.display = el.dataset.tariff === tariff ? 'block' : 'none';
});
```

**Канал 2: Email после оплаты (через Prodamus/ЮKassa)**

В настройках платёжной системы для каждого товара — автоматическое письмо с ссылкой на материалы. Страховка на случай, если человек закрыл вкладку или потерял ссылку.

Алексей настраивает тексты писем в Prodamus самостоятельно.

### Масштабирование

- 1 курс = 1 thank-you страница (параметр `tariff` переключает блоки)
- 5 курсов = 5 страниц (не 15)
- Для нового курса: скопировать страницу, заменить тексты и ссылки

### Технические требования

- Страница **НЕ индексируется** (noindex, nofollow)
- Без полной навигации — только логотип
- JS для переключения блоков по параметру `tariff`
