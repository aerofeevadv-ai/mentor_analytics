## 1. Правки на главной странице (/)

### 1.1 Меню — сделать «Курсы» выпадающим

Сейчас «Курсы» — якорная ссылка на #products.

Нужно: dropdown при наведении (hover).

| Пункт | Ссылка | Вид |
|-------|--------|-----|
| Продуктовые метрики и кейсы | /product-metrics | Обычная ссылка |
| A/B-тесты | — | Серый текст + бейдж «Скоро» |
| Road Map аналитика | — | Серый текст + бейдж «Скоро» |
| Все продукты | /#products | Обычная ссылка |

Бейдж «Скоро»: фон #e5e5e5, текст #666, скругление 8px, шрифт 12px.

На мобильных: подпункты внутри бургер-меню.

**Важно:** это же меню должно быть на всех страницах сайта (/, /product-metrics, /personal-mentor). Рекомендую использовать глобальный блок Tilda.

## 3. Новая страница: /thanks-product

Страница благодарности после успешной оплаты курса.

### Макет

Простая, минимальная. Белый фон, контент по центру. Шапка — только логотип «Mentor Analytics.», без меню.

### Содержание

```
[Зелёная галочка / иконка успеха]

Оплата прошла успешно!

Спасибо за покупку курса «Продуктовые метрики и кейсы».


[Кнопка, основная синяя: Перейти к курсу →]
← ссылка предоставлю отдельно


Что дальше:

1. Начни с модуля 1 — там введение и план прохождения
2. Рекомендую проходить по 1-2 урока в день
3. Вопросы по курсу: mentoranalytics@yandex.ru
4. Подпишись на канал — t.me/analytic_offers
```

### Технические требования

- URL: `mentoranalytics.ru/thanks-product`
- Redirect URL для платёжки (Prodamus/ЮKassa) → эта страница
- **noindex, nofollow** — не индексировать
- Без меню навигации, только логотип

## 4. Новая страница: /links (link-in-bio)

Страница для bio соцсетей (Instagram, VK, TikTok, YouTube).

### Макет

**Mobile-first.** На десктопе — узкая колонка по центру (макс 480px).
**Без шапки и футера основного сайта** — самостоятельная страница.

### Содержание

```
[Фото — круглое, 120px]
(то же фото, что в блоке «Обо мне» на главной)

Алексей Ерофеев
Ментор аналитиков | ex-Яндекс

Аналитика без воды — карьера, метрики, собесы


[Кнопки-ссылки, вертикально, полная ширина]
```

### Кнопки

| # | Текст | Ссылка | Стиль |
|---|-------|--------|-------|
| 1 | Курс: Продуктовые метрики и кейсы | /product-metrics | Акцентная: синий фон #4a6fff, белый текст |
| 2 | Telegram-канал | https://t.me/analytic_offers | Обычная: белый фон, синий бордер |
| 3 | YouTube | https://youtube.com/@analytic_offers | Обычная |
| 4 | Instagram | (ссылку предоставлю) | Обычная |
| 5 | VK | (ссылку предоставлю) | Обычная |

Стиль кнопок: полная ширина, высота ~56px, скругление 16px, Inter 500 16px.

### Подвал

```
mentoranalytics.ru
```
Ссылка на главную, мелким серым текстом по центру.

### Дизайн

- Фон: белый (#fff) или #fafafa
- Без навигации сайта
- Без стандартного футера
- Лёгкий fade-in кнопок при загрузке (опционально)
