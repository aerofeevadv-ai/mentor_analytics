# Python + Pandas для аналитиков: 20/80 — Структура

## Цель курса

После прохождения ученик:
- решает pandas-задачи уровня собеседований junior/middle (фильтрация, группировки, merge, пропуски, строки) быстро и без подсказок
- уверенно пишет базовый Python: коллекции, условия, циклы, функции; читает чужой код
- решает алгоритмические задачи уровня собесов аналитиков: проход по массиву, два указателя
- решает типовые рабочие задачи аналитика: retention, воронки, ARPU, когорты
- имеет личную базу решённых ноутбуков как опору на испытательном сроке

## Аудитория

1. Новички: SQL уже знают, Python с нуля
2. Работающие аналитики со слабым Python (заходят сразу в модуль 2)

## Принцип

20/80 + автоматизация навыка: не «прочитал и понял», а «повторил до автоматизма».
Ежедневный практикум — ядро курса, не дополнение.

На собеседованиях LLM запрещён, поэтому весь тренировочный трек решается руками.
LLM — инструмент обучения (объяснить ошибку, разложить решение), не решатель.

Аудитория знает SQL → параллели pandas ↔ SQL встроены в карточки методов
(groupby = GROUP BY, merge = JOIN), полная шпаргалка в финале.

## Модули

### 0. Стартовый блок
Онбординг: как устроен курс, темп (3–4 недели), настройка окружения (Anaconda + Jupyter),
как учиться с LLM, FAQ и техподдержка. Две точки входа: новичок / работающий аналитик.

### 1. Минимальный Python
Типы данных (mutable/immutable), списки, словари, множества, кортежи,
условия, циклы, функции. Критические основы (отступы, = vs ==, чтение ошибок).
Практика 3 уровней (лёгкий / средний / сложный).
Видео по теории не записываем: основа — конспекты, чужой YouTube-ролик остаётся
опциональной ссылкой «если хочешь лекционный формат».

### 2. Pandas: фундамент и ключевые методы
DataFrame/Series, загрузка данных (CSV/Excel/SQL-концепт), первичный анализ,
векторное мышление, индекс, пропуски и типы.
Затем 10 групп ключевых методов (формат карточек: что делает / простой пример /
сложнее / подводные камни / аналог в SQL): просмотр, фильтрация, колонки,
groupby/agg/pivot, merge/concat, пропуски, дубликаты, сортировка, строки, чтение/запись.

### 3. Практика до автоматизма
- Ежедневный практикум: 20 задач, повторять 5–7 дней

  здоровье, маркетинг) + задачи. Датасеты генерим скриптами: контролируемые
  пропуски, дубли, кривые типы — под учебные задачи
- Рабочие задачи junior/middle: retention, воронки, ARPU, когорты
- Мини-визуализация: matplotlib для метрик (line, bar, hist) — минимум

### 4. Собеседование и финал
- Как проходит Python-секция на собесе аналитика

### 5. Алгоритмы для собеседований (расширенный тариф)
Покрывает ~90% алго-вопросов на собесах аналитиков:
- Сложность алгоритмов: Big-O на пальцах, без математики (O(1), O(n), O(n²), O(n log n))
- Проход по массиву: max/min, суммы, частоты через dict, уникальность через set
- Два указателя: пары с заданной суммой, палиндром, слияние отсортированных
- Формат решения на собесе: функция solve(), проговаривание вслух, edge cases
- Тренировочный набор задач с решениями и разбором

### 6. Записи реальных собеседований (расширенный тариф)
Анонимизированные записи Python-секций собеседований + конспекты-разборы:
вопросы, решение кандидата, ошибки, как ответить сильнее.
Формат и pipeline — как в продуктовом курсе (process-interview).
