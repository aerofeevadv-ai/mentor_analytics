# 🎥 Модуль 6. Записи реальных собеседований 🔒

Анонимизированные записи Python-секций реальных собеседований аналитиков — с разборами: где кандидат ошибся, что сделал сильно, как ответить лучше.

---

## Как проходить этот модуль

1. **Сначала реши сам.** Перед каждой записью вынесены задачи с собеса — реши их до просмотра
2. **Смотри запись активно:** ставь паузу после вопроса интервьюера и отвечай вслух, потом сравнивай с кандидатом
3. **Читай разбор** и отмечай в своём списке слабых мест (трекер из [3.1](../module_3/3_1_methodology.md)), что прокачать

> 💡 **Главная мысль:** запись чужого собеса — тренажёр без риска. Ошибки кандидата запоминаются лучше своих, потому что смотришь со стороны и без стресса.
