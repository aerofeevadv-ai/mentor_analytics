# 2.2 Загрузка данных

Работа аналитика начинается с данных. Важный навык — загрузить правильно, понять структуру и быстро починить ошибки формата.

---

## CSV — основной формат в обучении и обмене данными

CSV — текстовый файл, значения разделены символом (запятая или точка с запятой).

```python
import pandas as pd

df = pd.read_csv("data.csv")
```

**Два параметра, которые должен знать каждый:**

```python
# sep — разделитель. ";" очень част в российских выгрузках
df = pd.read_csv("data.csv", sep=";")

# encoding — кодировка. Если русский текст превратился в "????"
df = pd.read_csv("data.csv", encoding="utf-8")
df = pd.read_csv("data.csv", encoding="cp1251")   # типично для Windows-выгрузок
```

**Реже, но полезно:**

```python
df = pd.read_csv("big.csv", nrows=1000)                      # первые N строк большого файла
df = pd.read_csv("data.csv", usecols=["id", "age", "city"])  # только нужные колонки
```

### ⚠️ Частые ошибки загрузки CSV

| Ошибка | Причина | Решение |
|---|---|---|
| `FileNotFoundError` | файл не в той папке | положи CSV рядом с ноутбуком, проверь имя 1-в-1 |
| `UnicodeDecodeError` | кодировка | пробуй utf-8 → cp1251 → latin1 |
| `ParserError` | неверный разделитель | открой файл блокнотом, посмотри разделитель, задай sep |

## SQL — главный источник данных в реальной работе

В компании рабочий процесс почти всегда такой:

1. Пишешь SQL-запрос к базе
2. Получаешь результат как DataFrame
3. Анализируешь его в pandas

```python
import pandas as pd
import sqlalchemy

engine = sqlalchemy.create_engine("postgresql://user:pass@host:5432/db")
df = pd.read_sql("SELECT * FROM orders WHERE dt >= '2026-01-01'", engine)
```

> 💡 **Главная мысль:** SQL и pandas не конкуренты, а конвейер. SQL достаёт и предагрегирует данные из базы, pandas обрабатывает их в ноутбуке: метрики, когорты, графики. Ты уже умеешь первую половину конвейера — этот курс достраивает вторую.

Подключение к базам настраивать здесь не будем — в каждой компании оно своё, а синтаксис pandas от источника не меняется.

🎯 **Мини-практика:** возьми любой CSV (или сохрани таблицу из прошлой практики через `df.to_csv("test.csv", index=False)`) и загрузи его: целиком, первые 2 строки через nrows, только одну колонку через usecols.
