# 🐼 Модуль 2. Pandas: фундамент и ключевые методы

Главный модуль курса. Сначала фундамент — как устроен DataFrame и как «мыслить векторно». Потом 10 групп методов, которые покрывают 80% рабочих задач аналитика.

> 📌 **Контекст:** pandas — стандартный инструмент аналитики в каждой компании. Задачи на pandas дают на 30–40% технических собеседований — чаще, чем что-либо другое по Python.

> 💡 Ты знаешь SQL — это твоё преимущество. У каждого метода в этом модуле указан SQL-аналог: половину концепций ты уже знаешь, осталось выучить новый синтаксис.

## Датасеты модуля

Все практики используют датасеты из папки `datasets/csv/` (архив прилагается к курсу, ссылка на скачивание — в [модуле 3](../module_3/00_index.md)). Основной файл — `data.csv` (518 строк, 8 колонок: user_id, name, age, city, salary, email, segment, signup_date). Для практик по merge используются `orders.csv` и `users.csv` из той же папки.

## Легенда меток

- **★** — опциональный материал для углублённого изучения. Не критично для собеседований базового уровня.
- Всё без ★ — обязательный минимум 20/80.

📌 Применение к методам: `between()` в B2 помечено ★ — знать приятно, но без него фильтрация работает. `apply()` и `map()` в B3 — обязательны, ★ снят: частые темы собеседований.
