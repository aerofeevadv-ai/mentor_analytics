## Задания

**1. Импорт.** Импортируй pandas под стандартным псевдонимом.

**2. Загрузка.** Загрузи `data.csv` в `df`. Выведи первые 5 строк, затем первые 7. Попробуй загрузить с `sep=";"` и `encoding="utf-8"` — посмотри, что изменится.

**3. Структура.** Выведи: последние 5 строк, 3 случайные строки, структуру таблицы, статистику числовых колонок, размеры датасета.

**4. Типы и пропуски.** Посмотри типы колонок, булеву маску пропусков, количество пропусков по колонкам.

**5. Колонки.** Выбери колонку `age`; выбери `name` и `city` вместе; создай `age_plus_1`; создай булеву `is_adult` (возраст ≥ 18); перезапиши `age` умножением на 2.

**6. Фильтрация.** Строки, где возраст больше 30; где город равен «Moscow»; где зарплата не пропуск.

**7. Индекс.** Сбрось индекс с `drop=True`, убедись, что нумерация начинается с нуля.

<summary>✅ Решение</summary>

```python
# 1. Импорт
import pandas as pd

# 2. Загрузка
df = pd.read_csv("data.csv")
df.head()
df.head(7)
# df = pd.read_csv("data.csv", sep=";")
# df = pd.read_csv("data.csv", encoding="utf-8")

# 3. Структура
df.tail()
df.sample(3)
df.info()
df.describe()
df.shape

# 4. Типы и пропуски
df.dtypes
df.isna()
df.isna().sum()

# 5. Колонки
df["age"]
df[["name", "city"]]
df["age_plus_1"] = df["age"] + 1
df["is_adult"] = df["age"] >= 18
df["age"] = df["age"] * 2

# 6. Фильтрация
df[df["age"] > 30]
df[df["city"] == "Moscow"]
df[df["salary"].notna()]

# 7. Индекс
df = df.reset_index(drop=True)
df.head()
```
