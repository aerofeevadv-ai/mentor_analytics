# 3.9 Мини-визуализация

Минимум matplotlib, который покрывает 90% потребностей аналитика: линия, столбцы, гистограмма. Без зоопарка графиков.

---

## Три графика, которые нужны всегда

```python
import matplotlib.pyplot as plt
```

**Line — динамика во времени:**

```python
weekly = clean.groupby(clean["order_date"].dt.to_period("W"))["revenue"].sum()

weekly.plot(kind="line", figsize=(10, 4), title="Выручка по неделям")
plt.xlabel("Неделя")
plt.ylabel("Выручка, ₽")
plt.show()
```

**Bar — сравнение категорий:**

```python
by_city = clean.groupby("city")["revenue"].sum().sort_values(ascending=False)

by_city.plot(kind="bar", figsize=(8, 4), title="Выручка по городам")
plt.ylabel("Выручка, ₽")
plt.show()
```

**Hist — распределение:**

```python
clean["revenue"].plot(kind="hist", bins=50, figsize=(8, 4),
                      title="Распределение чека")
plt.xlabel("Сумма заказа, ₽")
plt.show()
```

---

## Какой график когда

| Вопрос | График |
|---|---|
| Как метрика меняется во времени? | line |
| Что больше / меньше по категориям? | bar |
| Как распределены значения, где выбросы? | hist |

> ⚠️ **Типичная ошибка:** pie chart для сравнения 8 категорий. Глаз не сравнивает углы — сравнивает длины. Используй bar, отсортированный по убыванию.

## Минимальные требования к графику

Любой график, который ты показываешь другому человеку:

- заголовок: что на графике
- подписи осей с единицами измерения
- разумный размер (`figsize=(10, 4)`)
- bar отсортирован по значению, line — по времени

Этого хватит. Цвета, стили и интерактив — не сейчас.

> 💡 **Главная мысль:** график строится не «чтобы был», а чтобы ответить на вопрос. Перед построением скажи, на какой вопрос отвечаешь, после — что видно. Если словами сказать нечего, график не нужен.
