# 🐍 Модуль 1. Минимальный Python

Минимальный, но прочный фундамент Python перед pandas. Это не курс программирования — это набор навыков, которые аналитик реально использует в работе.

> 💡 **Главная мысль:** Python не должен быть сложным — он должен быть утилитарным. Загрузить → почистить → преобразовать → проанализировать. ООП, классы и декораторы аналитику не нужны, их здесь нет.

## Критерий готовности к pandas

Переходи к модулю 2, если:

- понимаешь разницу между изменяемыми и неизменяемыми объектами
- уверенно работаешь со списками, строками, словарями, множествами
- понимаешь условия, циклы и функции
- читаешь и пишешь простые comprehensions, сортируешь через `sorted(key=...)`
- решаешь практику уровней 1–2 без подсказок
