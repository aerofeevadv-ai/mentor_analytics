# 1.7 Циклы (for / while)

Циклы повторяют действие: для каждого элемента коллекции или пока выполняется условие.

> 📌 **Контекст:** в работе аналитика циклы для обработки данных используются редко — это делает pandas векторно. Но циклы обязательны для понимания, как всё работает под капотом, и для алгоритмических задач на собеседованиях, где pandas нет.

---

## Цикл for — перебор коллекции

Самый используемый цикл. Перебирает списки, словари, множества, строки, диапазоны.

```python
# Основной шаблон
numbers = [10, 20, 30]

for n in numbers:
    print(n)
# 10
# 20
# 30
```

**Формирование нового списка** — базовый паттерн:

```python
numbers = [10, 20, 30]
result = []

for n in numbers:
    result.append(n * 2)

print(result)   # → [20, 40, 60]
```

Запомни тройку: создал пустой список → перебрал → добавил подходящее. На этом паттерне держится половина задач уровня junior.

**Перебор с индексом — enumerate:**

```python
for i, n in enumerate(numbers):
    print(i, n)
# 0 10
# 1 20
# 2 30
```

**Перебор по индексам — range(len()):**

```python
for i in range(len(numbers)):
    print("Индекс:", i, "| Значение:", numbers[i])
```

Нужен, когда важна позиция: сравнить соседние элементы, изменить элементы на месте. Часто встречается на собеседованиях.

**Попарный проход двух списков — zip:**

```python
names = ["Bob", "Ann", "Tom"]
ages = [25, 31, 19]

for name, age in zip(names, ages):
    print(name, age)
# Bob 25
# Ann 31
# Tom 19
```

`zip` идёт по спискам синхронно: первый с первым, второй со вторым. Если списки разной длины — остановится на коротком. Два бонусных приёма: `list(zip(a, b))` склеивает списки в список пар, а `dict(zip(keys, values))` строит словарь-маппинг из двух списков.

**Перебор словаря:**

```python
user = {"id": 1, "city": "Москва", "age": 25}

for key, value in user.items():
    print(key, value)
```

## range() — диапазоны чисел

```python
range(5)          # 0, 1, 2, 3, 4 — от нуля до stop (не включая)
range(2, 6)       # 2, 3, 4, 5 — от start до stop
range(10, 0, -2)  # 10, 8, 6, 4, 2 — с шагом, в том числе отрицательным
```

> ⚠️ **Типичная ошибка:** `range(5)` не включает 5. Та же логика «до, не включая», что и в срезах списков.

## Цикл while — пока условие истинно

```python
x = 0
values = []

while x < 3:
    values.append(x)
    x += 1

print(values)   # → [0, 1, 2]
```

Разбор по шагам:

1. `x = 0` → условие True → добавили 0 → `x = 1`
2. `x = 1` → условие True → добавили 1 → `x = 2`
3. `x = 2` → условие True → добавили 2 → `x = 3`
4. `x = 3` → условие False → цикл завершён

> ⚠️ **Типичная ошибка:** забыть изменить переменную условия (`x += 1`) — цикл станет бесконечным, и Jupyter зависнет. Если завис: Kernel → Interrupt.

## break и continue — управление циклом

**break — выйти из цикла досрочно:**

```python
for n in [3, 12, 5, 18, 1, 20]:
    if n > 15:
        break          # встретили 18 — выходим, до 1 и 20 дело не дойдёт
    print(n)
# 3
# 12
# 5
```

**continue — пропустить итерацию и перейти к следующей:**

```python
for n in [3, 12, 5, 18, 1, 20]:
    if n < 10:
        continue       # маленькие пропускаем
    print(n)
# 12
# 18
# 20
```

Когда что: `break` — «нашёл, что искал, дальше перебирать незачем»; `continue` — «это значение не подходит, идём дальше» (типично при чистке грязных данных).

> ⚠️ **Типичная ошибка:** `break` выходит только из ближайшего цикла. Если циклов два вложенных, внешний продолжит работать.

## Когда что использовать

| Ситуация | Инструмент |
|---|---|
| Перебрать элементы списка / строки / словаря | for |
| Пройти диапазон чисел | for + range |
| Повторять, пока условие истинно | while |
| Обработать колонку таблицы | **pandas, не цикл** |

> 💡 **Главная мысль:** увидел задачу «обработать каждую строку таблицы» — это pandas, не цикл. Цикл по строкам DataFrame — антипаттерн, который выдаёт новичка и работает в сотни раз медленнее. В модуле 2 увидишь, как то же самое делается одной векторной операцией.

🎯 **Мини-практика:** дан список `[3, 12, 5, 18, 1, 20]`. Через for собери новый список только из чисел больше 10. Затем через enumerate выведи каждый элемент с его индексом. Затем выводи элементы, пока не встретишь число больше 15 (подойдёт for + break). И последнее: через zip выведи пары из `names = ["Bob", "Ann"]` и `salaries = [100, 120]`.
