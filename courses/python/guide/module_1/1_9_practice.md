## Уровень 1 — лёгкий

**1. Создание переменных.** Создай переменные: `age` (число), `name` (строка), `height` (float), `is_active` (bool). Выведи их на экран.

**2. Типы данных.** Выведи тип каждой переменной с помощью `type()`.

**3. Арифметика.** Даны `a = 7`, `b = 3`. Вычисли: сумму, разницу, произведение, целочисленное деление, остаток, степень `a**b`.

**4. Индексация и срезы.** Дан `nums = [5, 10, 15, 20, 25]`. Получи: первый элемент, последний, срез с 1 по 3 индекс, длину списка.

**5. Изменение списка.** Добавь 30 в конец, удали число 10, вставь 7 на индекс 1.

**6. Словари.** Создай `user = {"name": "Alex", "age": 25, "city": "Moscow"}`. Измени age на 26, добавь ключ `"is_active": True`, удали ключ `"city"`.

**7. Множества.** Дан `vals = [1, 2, 2, 3, 4, 4, 5]`. Получи множество уникальных значений.

**8. Условия.** Если `age >= 18` — выведи `"adult"`, иначе `"minor"`.

**9. Цикл for.** Выведи все элементы списка `names = ["Bob", "Ann", "Tom"]`.

**10. Функция.** Напиши `greet(name)`, возвращающую строку `"Hello, <name>"`.

<details>
<summary>✅ Решения уровня 1</summary>

```python
# 1. Создание переменных
age = 25
name = "Alex"
height = 1.82
is_active = True
print(age, name, height, is_active)

# 2. Типы данных
print(type(age))
print(type(name))
print(type(height))
print(type(is_active))

# 3. Арифметика
a, b = 7, 3
print(a + b)    # сумма
print(a - b)    # разница
print(a * b)    # произведение
print(a // b)   # целочисленное деление
print(a % b)    # остаток
print(a ** b)   # степень

# 4. Индексация и срезы
nums = [5, 10, 15, 20, 25]
print(nums[0])      # первый
print(nums[-1])     # последний
print(nums[1:4])    # срез
print(len(nums))    # длина

# 5. Изменение списка
nums.append(30)
nums.remove(10)
nums.insert(1, 7)
print(nums)

# 6. Словари
user = {"name": "Alex", "age": 25, "city": "Moscow"}
user["age"] = 26
user["is_active"] = True
del user["city"]
print(user)

# 7. Множества
vals = [1, 2, 2, 3, 4, 4, 5]
unique_vals = set(vals)
print(unique_vals)

# 8. Условия
if age >= 18:
    print("adult")
else:
    print("minor")

# 9. Цикл for
names = ["Bob", "Ann", "Tom"]
for n in names:
    print(n)

# 10. Функция
def greet(name):
    return "Hello, " + name

print(greet("Alex"))
```

## Уровень 2 — средний

**1. Квадраты.** Дан `[1, 2, 3, 4, 5]`. Создай новый список квадратов чисел.

**2. Фильтрация.** Из `[3, 12, 5, 18, 1, 20]` оставь только элементы больше 10.

**3. Минимум и максимум вручную.** Из `nums = [14, 2, 9, 27, 1]` найди минимум и максимум **без** `min()`/`max()`.

**4. Разворот вручную.** Получи обратный список без `.reverse()` и срезов.

**5. Частоты.** Из `["a", "b", "a", "c", "b", "a"]` создай словарь частот: `{"a": 3, "b": 2, "c": 1}`.

**6. Фильтрация словаря.** Из `{"a": 5, "b": 15, "c": 2, "d": 20}` выбери только пары со значением больше 10.

**7. Перебор через range(len()).** Для `[10, 20, 30]` выведи `index: 0 -> 10` и так далее.

<details>
<summary>✅ Решения уровня 2</summary>

```python
# 1. Квадраты
nums = [1, 2, 3, 4, 5]
squares = []
for n in nums:
    squares.append(n**2)
print(squares)

# 2. Фильтрация
nums = [3, 12, 5, 18, 1, 20]
filtered = []
for n in nums:
    if n > 10:
        filtered.append(n)
print(filtered)

# 3. Минимум и максимум вручную
nums = [14, 2, 9, 27, 1]
mn, mx = nums[0], nums[0]
for n in nums:
    if n < mn:
        mn = n
    if n > mx:
        mx = n
print(mn, mx)

# 4. Разворот списка вручную
nums = [1, 2, 3, 4, 5]
rev = []
for i in range(len(nums) - 1, -1, -1):
    rev.append(nums[i])
print(rev)

# 5. Частоты
data = ["a", "b", "a", "c", "b", "a"]
freq = {}
for x in data:
    if x not in freq:
        freq[x] = 1
    else:
        freq[x] += 1
print(freq)

# 6. Фильтрация словаря
d = {"a": 5, "b": 15, "c": 2, "d": 20}
filtered = {}
for k, v in d.items():
    if v > 10:
        filtered[k] = v
print(filtered)

# 7. range(len())
nums = [10, 20, 30]
for i in range(len(nums)):
    print("index:", i, "->", nums[i])
```

</details>

## Уровень 3 — сложный (опциональный)

Если хочешь дополнительный вызов. Эти паттерны вернутся в модуле алгоритмов.

**1. Чётные числа.** Из `[1, 2, 3, 4, 5, 6, 7, 8]` получи только чётные.

**2. Второй максимум.** `[10, 5, 8, 20, 3]` → `10`.

**3. Количество пар одинаковых чисел.** `[1, 2, 1, 2, 1, 5]` → 2 пары (пара единиц + пара двоек).

**4. Словарь «значение → индекс».** `[10, 20, 30]` → `{10: 0, 20: 1, 30: 2}`.

**5. Порог.** Удали все элементы меньше порога `t = 10` из `[3, 12, 7, 18, 1, 20]`.

**6. Попарное объединение.** `a = [1, 2, 3]`, `b = [4, 5, 6]` → `[(1, 4), (2, 5), (3, 6)]`.

<details>
<summary>✅ Решения уровня 3 (с построчным разбором)</summary>

```python
# 1. Чётные числа
nums = [1, 2, 3, 4, 5, 6, 7, 8]
evens = []                       # пустой список для чётных

for n in nums:                   # перебираем каждый элемент
    if n % 2 == 0:               # делится на 2 без остатка?
        evens.append(n)          # да — добавляем

print(evens)

# 2. Второй максимум
nums = [10, 5, 8, 20, 3]

mx = nums[0]                     # ищем реальный максимум
for n in nums:
    if n > mx:
        mx = n

second = None                    # переменная для второго максимума
for n in nums:
    if n != mx:                  # пропускаем сам максимум
        if second is None or n > second:
            second = n           # обновляем, если больше текущего

print(second)

# 3. Количество пар одинаковых чисел
arr = [1, 2, 1, 2, 1, 5]
freq = {}                        # число → сколько раз встретилось

for x in arr:
    if x not in freq:
        freq[x] = 1
    else:
        freq[x] += 1

pairs = 0
for v in freq.values():
    pairs += v // 2              # каждая пара — 2 одинаковых элемента

print(pairs)

# 4. Словарь "значение → индекс"
lst = [10, 20, 30]
d = {}

for i, v in enumerate(lst):      # enumerate даёт (индекс, значение)
    d[v] = i

print(d)

# 5. Порог
nums = [3, 12, 7, 18, 1, 20]
t = 10
res = []

for n in nums:
    if n >= t:                   # оставляем только >= порога
        res.append(n)

print(res)

# 6. Попарное объединение
a = [1, 2, 3]
b = [4, 5, 6]
pairs = []

for i in range(len(a)):          # перебираем индексы
    pairs.append((a[i], b[i]))   # пара из элементов с одним индексом

print(pairs)
```

</details>
