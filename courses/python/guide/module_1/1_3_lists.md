# 1.3 Списки (list)

**Тип: изменяемый (mutable)**

Список — упорядоченная последовательность элементов. Главная коллекция Python и концептуальный предок Series в pandas.

---

## Что значит «упорядоченная»

- порядок сохраняется: `[10, 20, 30]` — всегда сначала 10, потом 20, потом 30
- у каждого элемента есть **индекс** (позиция), нумерация с нуля
- элементы могут повторяться и быть разных типов

```python
numbers = [1, 5, 10]
mixed = [10, "abc", True, None]   # разные типы — допустимо
```

---

## Индексация и срезы

```python
numbers = [5, 10, 15, 20, 25]

numbers[0]     # первый элемент → 5
numbers[2]     # третий → 15
numbers[-1]    # последний → 25

numbers[1:3]   # срез: индексы 1 и 2 → [10, 15]
numbers[:2]    # от начала до индекса 2 (не включая) → [5, 10]
numbers[2:]    # от индекса 2 до конца → [15, 20, 25]

len(numbers)   # длина → 5
```

> ⚠️ **Типичная ошибка:** срез `[1:3]` НЕ включает правую границу. Берутся индексы 1 и 2, но не 3. Правило: «от включительно, до не включая» — как BETWEEN в SQL работает иначе, не перепутай.

---

## Изменение списка

```python
numbers = [5, 10, 15]

# Добавить в конец
numbers.append(20)        # [5, 10, 15, 20]

# Добавить несколько
numbers.extend([30, 40])  # [5, 10, 15, 20, 30, 40]

# Вставить по индексу
numbers.insert(1, 99)     # [5, 99, 10, 15, 20, 30, 40]

# Удалить первое вхождение значения
numbers.remove(10)        # ищет значение 10, не индекс

# Удалить по индексу
numbers.pop(0)            # удалил первый элемент

# Перезаписать по индексу
numbers[1] = 100
```

> ⚠️ **Типичная ошибка:** путать `remove(10)` (удаляет ЗНАЧЕНИЕ 10) и `pop(10)` (удаляет элемент с ИНДЕКСОМ 10).

---

## 🐼 Мост к pandas

> 💡 **Главная мысль:** Series в pandas ведёт себя как список с суперспособностями: тоже порядок, тоже индексы, тоже длина. Понимаешь списки — DataFrame перестаёт быть магией на 80%.

```python
# Список → Series — прямое родство
import pandas as pd
s = pd.Series([5, 10, 15])
```

🎯 **Мини-практика:** создай список из 5 чисел. Получи последний элемент двумя способами (через положительный и отрицательный индекс). Добавь число в конец, удали второе, выведи длину.
