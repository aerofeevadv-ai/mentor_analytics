# 1.4 Словари (dict)

**Тип: изменяемый (mutable)**

Словарь — структура «ключ → значение». Данные находятся по ключу мгновенно, без перебора.

---

## Как устроен

- каждый элемент — пара: уникальный **ключ** и его **значение**
- доступ по ключу: `user["city"]` — мгновенно
- ключи уникальны, порядок добавления сохраняется
- значения — любые: числа, строки, списки, другие словари

```python
user = {"id": 15, "city": "Москва", "age": 30}
```

Это буквально строка таблицы: имена колонок → значения.

---

## Операции

```python
user = {"id": 15, "city": "Москва", "age": 30}

# Получить значение
user["city"]              # → "Москва"

# Добавить ключ
user["is_active"] = True

# Изменить значение
user["age"] = 31

# Удалить ключ
del user["city"]

# Проверить наличие ключа
"id" in user              # → True

# Все ключи / значения / пары
list(user.keys())         # → ['id', 'age', 'is_active']
list(user.values())       # → [15, 31, True]
list(user.items())        # → [('id', 15), ('age', 31), ('is_active', True)]
```

> ⚠️ **Типичная ошибка:** обращение к несуществующему ключу — `user["salary"]` → KeyError. Если ключа может не быть, проверь через `in` или используй `user.get("salary")` — он вернёт None вместо ошибки.

---

## Классика собеседований: подсчёт частот

Словарь — стандартный инструмент для «посчитай, сколько раз встречается каждый элемент»:

```python
data = ["a", "b", "a", "c", "b", "a"]
freq = {}

for x in data:
    if x not in freq:
        freq[x] = 1
    else:
        freq[x] += 1

print(freq)   # → {'a': 3, 'b': 2, 'c': 1}
```

Напоминание: `freq[x] += 1` — это `freq[x] = freq[x] + 1`, оператор `+=` из раздела 1.2.

Запомни этот паттерн — он вернётся в практике, а в разделе [1.8b sorted + lambda](1_8b_sorted_lambda.md) мы достанем из словаря частот топ-N значений. Глубже тема разбирается в модуле алгоритмов (расширенный тариф).

## 🐼 Мост к pandas

> 💡 **Главная мысль:** словарь в аналитике — это маппинг: код города → название, ID категории → категория. В pandas ты будешь передавать словари в `map()` и `rename()`, создавать DataFrame из словаря списков. По смыслу это `JOIN` со справочником, только в миниатюре.

🎯 **Мини-практика:** создай словарь с тремя товарами (название → цена). Подними цену одного товара на 10%, добавь четвёртый товар, удали первый, выведи все пары через `items()`.
