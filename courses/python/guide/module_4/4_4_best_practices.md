# 4.4 Best practices: код, за который берут

Один и тот же результат можно получить кодом, который хочется читать, и кодом, который хочется закрыть. Интервьюер видит разницу мгновенно.

---

## Ноутбук как документ

- **Ячейка = шаг анализа.** Загрузка, чистка, метрика, вывод — отдельными ячейками, не простынёй
- **После каждого шага — контроль:** head(), shape, сумма. Ты показываешь, что проверяешь себя
- **Markdown-заголовки между блоками,** если задача длинная
- **Ноутбук выполняется сверху вниз** после Restart & Run All. Ячейки, прыгающие по порядку — красный флаг

---

## Имена и читаемость

```python
# ❌
df2 = df[df["a"] > 0]
x = df2.groupby("b")["c"].sum()

# ✅
paid_orders = orders[orders["price"] > 0]
revenue_by_city = paid_orders.groupby("city")["revenue"].sum()
```

- имя говорит, ЧТО внутри: `paid_orders`, а не `df2`
- длинные цепочки разбивай скобками по строкам:

```python
top_cities = (
    orders[orders["status"] == "completed"]
    .groupby("city")["revenue"]
    .sum()
    .sort_values(ascending=False)
    .head(5)
)
```

## Несколько правил-маркеров уровня

| Правило | Почему |
|---|---|
| Запись через loc, не chained assignment | см. [tricky №5](4_2_tricky_pandas.md) |
| Без inplace=True | явное присваивание читается и не ломает цепочки |
| index=False при to_csv | без мусорной Unnamed: 0 |
| Векторные операции вместо циклов и лишних apply | скорость + стиль |
| Сырые данные не перезаписываются | исходник свят, трансформации воспроизводимы кодом |
| Магические числа — в переменные с именем | `MIN_ORDER_SUM = 1000` |

---

## Чек-лист самопроверки перед показом кода

- [ ] Restart & Run All проходит без ошибок
- [ ] имена переменных читаются без расшифровки
- [ ] после merge/фильтров проверены размеры
- [ ] пропуски и дубли обработаны осознанно (могу объяснить почему так)
- [ ] результат сформулирован словами в последней ячейке

> 💡 **Главная мысль:** на испытательном сроке твой код будут читать коллеги. Интервьюер оценивает не красоту, а «смогу ли я это ревьюить и поддерживать». Пиши на собесе так, как писал бы в команде.
