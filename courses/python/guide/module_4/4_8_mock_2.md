## Задача 3. SQL → pandas

**Условие.** Перепиши этот SQL запрос на pandas:

```sql
SELECT channel,
       SUM(spend)   AS total_spend,
       SUM(revenue) AS total_rev,
       SUM(revenue) / SUM(spend) AS roas
FROM campaigns
GROUP BY channel
ORDER BY roas DESC;
```
