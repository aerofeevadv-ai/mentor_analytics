# 🐍 Python + Pandas для аналитиков: 20/80

Курс для аналитиков, которые хотят **уверенно решать задачи на Python и pandas** — на собеседованиях и в работе.

---

## О чём этот курс

Этот курс не про программирование ради программирования.

Он про **рабочий навык**: загрузить данные, почистить, посчитать метрики, ответить на вопрос бизнеса. И про то, чтобы делать это руками, быстро и без подсказок — так, как требуют на технических собеседованиях.

> 💡 **Принцип 20/80:** мы изучаем 20% возможностей Python и pandas, которые покрывают 80% реальных задач аналитика. Всё остальное — по мере необходимости, уже на работе.

---

## Почему именно это

Статистика технических собеседований аналитиков:

| Секция | Доля собеседований |
|---|---|
| Задачи на pandas | 30–40% |
| Алгоритмические задачи | ~20% |

В работе аналитика Python — это почти всегда **pandas + визуализация в Jupyter**. Именно этому и учим.

---

## Что ты умеешь после курса

### 🧠 Собеседования

- решаешь pandas-задачи: фильтрация, группировки, merge, пропуски, строки
- пишешь чистые функции и проговариваешь решение вслух
- не пугаешься tricky-вопросов: NaN, дубли при merge, копия vs view
- переводишь SQL-запрос в pandas и обратно

### 🛠 Работа

- уверенно работаешь в Jupyter: загрузка → чистка → анализ → вывод
- считаешь retention, воронки, ARPU, когорты на реальных данных
- строишь графики, которые можно показать продакту
- используешь LLM как ускоритель, а не костыль
