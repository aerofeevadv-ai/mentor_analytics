**Критические основы (must-know, первая страница)**
- Отступы вместо скобок — ошибка №1 новичков
- print() как главный инструмент отладки
- Комментарии, snake_case, чувствительность к регистру
- `=` vs `==` — топ-3 ошибка, с примером SyntaxError
- Чтение ошибок: NameError / KeyError / IndexError / TypeError — таблица «ошибка → причина»
- type(), код читается сверху вниз

**Типы данных**
- int / float / str / bool / None — по карточке с примером «зачем аналитику»
- Mutable vs immutable: почему это важно (копия vs изменение на месте, поведение DataFrame)

**Коллекции (по единому шаблону: что это / зачем аналитику / операции / связь с pandas)**
- list: индексация, срезы, append/extend/insert, remove/pop, изменение по индексу;
  мост «Series ведёт себя как список»
- dict: ключ→значение, добавление/изменение/удаление, in, keys/values/items;
  мост «маппинг категорий, строка таблицы»
- tuple: неизменяемость, когда нужен; мост «индексы в pandas»
- set: уникальность, & | -, ограничения; мост «drop_duplicates, проверка категорий»

**Управление потоком**
- if / elif / else, вложенные условия; мост «булевы маски в pandas»
- for: перебор списка/словаря, enumerate, range(len()) (встречается на собесах),
  формирование нового списка
- range(): три формы (stop / start,stop / step)
- while: шаблон, пошаговый разбор, как не уйти в бесконечный цикл
- Таблица «когда for / когда while / когда pandas вместо цикла»

**Функции**
- def, имя (PEP8 + паттерны get_/is_/calculate_/solve), аргументы и значения по умолчанию,
  return (и что без return будет None), возврат нескольких значений
- Практический пример normalize_price (валидация → округление → минимум)
- Почему на собесе ждут функцию: чистый интерфейс, тестируемость
- Признаки хорошей функции (чек-лист из 5 пунктов)

**Закрытие модуля**
