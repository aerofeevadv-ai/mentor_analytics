# Модуль 0: Стартовый блок

## Ключевые тезисы

**Приветствие и цель**
- Курс про навык, а не теорию: решать pandas-задачи на собесах и в работе
- Статистика собесов как мотивация: pandas 30–40% собеседований, алгоритмы ~20%
- Принцип 20/80: минимальная теория → максимум повторяемой практики

**Как устроен курс**
- Карта модулей 0–6, что в базовом и что в расширенном тарифе
- Две точки входа: новичок идёт последовательно с модуля 1;
  работающий аналитик проходит самопроверку (5 вопросов) и заходит сразу в модуль 2
- Темп: 3–4 недели, 30–60 минут в день. Быстрее можно, но навык строится повторением
- Правила: практика важнее чтения, решения не подглядывать, свои ноутбуки сохранять —
  это личная база знаний на испытательный срок

**Как учиться с LLM**
- LLM (ChatGPT/Claude) — наставник, не решатель: «объясни ошибку», «разложи решение по шагам»,
  «дай похожую задачу», «проверь мой код»
- Жёсткое правило: задачи тренировочного трека решаем руками — на собесе LLM запрещён
- Готовые промпты для учёбы (страница-шпаргалка)

**Настройка окружения**
- Anaconda + Jupyter Notebook: почему так (стандарт аналитиков), шаги установки
  для Windows/macOS со скриншотами
- Первый ноутбук: создать, выполнить ячейку, Shift+Enter
- 5 частых ошибок установки и решения (из драфта: не запускается, Permission denied,
  FileNotFoundError, ModuleNotFoundError, не та папка)
- Локальный vs облачный Jupyter (в компаниях чаще облачный, функционально то же)
- Jupyter vs IDE: коротко, почему аналитику Jupyter
- Git: 2 абзаца — что это, зачем аналитику (DAG-и, ревью), для курса не нужен

**FAQ**
- Нужен ли SQL → ты его уже знаешь, будем опираться
- Нужен ли глубокий Python → нет, модуль 1 покрывает всё
- Техподдержка: куда писать
