# Модуль 6: Записи реальных собеседований (расширенный тариф)

## Ключевые тезисы

Формат и pipeline — как в продуктовом курсе (module_3/interviews, skill process-interview):
транскрибация → анонимизация → нарезка → конспект-разбор.

**Что входит**
- Анонимизированные записи Python-секций реальных собеседований аналитиков
- К каждой записи конспект: список вопросов → решение кандидата → где ошибся /
  что сделал хорошо → как ответить сильнее → ссылки на разделы курса по теме

**Как проходить (методика)**
- Сначала решить задачи из собеса самостоятельно (условия вынесены перед видео)
- Потом смотреть запись и сравнивать со своим решением
- В конце читать разбор

**Контент-вопросы (решить до написания)**
- ⚠️ есть ли уже записи собесов с Python-секциями в архиве, или собираем с нуля
  через учеников менторства?
- Минимум для запуска тарифа: 2–3 записи, дальше пополняемая библиотека
- Пополнение — операционный процесс: каждый новый собес ученика с Python-секцией →
  через process-interview → в курс

## Практика

- Задачи из каждого собеса — решить до просмотра записи
