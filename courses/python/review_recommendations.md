Не обещать «80% всех собеседований»: кроме Python там SQL, продукт, статистика, коммуникация.

## Принятые решения

- **Алгоритмы остаются в расширенном тарифе.** В базу переносится только
  `sorted(key=...)` / топ-N частот (модуль 1) — без него базовый тариф не может
  довести «классику частот» до собес-формата.
- **Моки — формат самопроверки**: таймер + задачи без решений на странице +
  чек-лист самооценки + решения отдельно. Живого проверяющего у внешнего
  покупателя курса нет.
- **Модуль 6 не блокирует запуск базы.** Помечается «пополняется», расширенный
  тариф не продавать до 2–3 обработанных записей.
