# Сценарий: Скринкаст «Jupyter за 10 минут»

**Тип:** скринкаст
**Хронометраж:** ~10 минут
**Раздел курса:** Модуль 0, страница 0.2 «Настройка окружения»

---

## До записи

- [ ] Чистая система или удалённая Anaconda (показываем установку с нуля)
- [ ] Браузер с открытой anaconda.com/download
- [ ] Файл data.csv лежит в Downloads (покажем перенос в рабочую папку)
- [ ] Масштаб экрана крупный, шрифт терминала/браузера увеличен
- [ ] Записать на macOS ИЛИ Windows; для второй системы — врезки скриншотов с отличиями

---

## Сценарий

[00:00] **Что сделаем**

За десять минут поставим всё окружение: Python, Jupyter и библиотеки. Ничего настраивать руками не будем — всё придёт одним установщиком.

[00:30] **Скачивание Anaconda**

Показываю: захожу на anaconda.com/download, выбираю свою систему, качаю. Файл большой, около гигабайта — пока качается, расскажу, что внутри: сам Python, Jupyter Notebook и все библиотеки для анализа данных — pandas, numpy, matplotlib. Поэтому и ставим Anaconda, а не голый Python: один установщик вместо десяти команд в терминале.

[01:30] **Установка**

Запускаю установщик, везде жму «далее» с настройками по умолчанию. Единственное важное: путь установки без русских букв. Если у тебя имя пользователя на кириллице — смотри заметку под видео.

[02:30] **Создаём рабочую папку**

Пока ставится: создаю папку PandasCourse в домашней директории. Все ноутбуки и файлы курса будут жить тут. Правило: никаких русских букв и пробелов в пути — сэкономишь себе пару ошибок FileNotFoundError.

[03:00] **Запуск Jupyter**

Открываю Anaconda Navigator, нахожу карточку Jupyter Notebook, жму Launch. Открылась вкладка в браузере — это и есть рабочий инструмент. Перехожу в папку PandasCourse.

[04:00] **Первый ноутбук**

New → Python 3. Переименовываю Untitled в first_steps. Вот ячейка — сюда пишем код, Shift+Enter — выполняем.

Пишу: print("привет"). Выполнил. Работает.

Теперь главная проверка: import pandas as pd. Выполнил — ошибок нет, pandas на месте.

[05:30] **Первая таблица**

Создаю маленький DataFrame прямо в ячейке (словарь → таблица), показываю вывод. Обрати внимание: последняя строка ячейки выводится сама, print не нужен.

[06:30] **Загрузка CSV**

Переношу data.csv из Downloads в папку PandasCourse — показываю мышкой. Файл должен лежать рядом с ноутбуком, это важно.

В ячейке: df = pd.read_csv("data.csv"), потом df.head(). Вот наша таблица.

И сразу показываю самую частую ошибку: пишу read_csv("dataa.csv") с опечаткой — ловлю FileNotFoundError. Читаем последнюю строку ошибки: нет такого файла. Проверь имя и где лежит файл. Девяносто процентов проблем первой недели — вот это.

[08:30] **Мини-навигация по Jupyter**

Самое нужное: Shift+Enter — выполнить ячейку, B — новая ячейка снизу, DD — удалить ячейку. Если ноутбук завис или ведёт себя странно: Kernel → Restart & Run All — перезапуск с чистого листа.

[09:30] **Итог**

Всё: окружение стоит, первая таблица загружена. Если что-то пошло не так — под видео список пяти частых ошибок установки с решениями. Дальше — модуль один, минимальный Python. Погнали!
