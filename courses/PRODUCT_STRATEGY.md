# Продуктовая стратегия: курсы и программы Mentor Analytics

> Дата: март 2026
> Статус: рабочий документ

---

## 1. Общая концепция

Курсы выполняют двойную роль:
1. **Внутреннюю** — входят в методологию менторства «до оффера» (ученики получают бесплатно)
2. **Внешнюю** — независимые продукты, приносящие прибыль вне привязки к менторству

Курсы **не являются трипваерами** к менторству. Каждый продукт самоценен. В единичных случаях покупатель курса может прийти на менторство — это побочный эффект, не цель.

## 3. Продуктовая линейка

### 3.1 Миникурс: Продуктовая аналитика

**Позиционирование:** «20% знаний, с помощью которых ты закроешь 80% собесов и рабочих задач.»

**ЦА:** аналитики с опытом 0.5–2 года, хотят в продуктовую аналитику.

| Тариф | Цена | Состав |
|-------|------|--------|
| Базовый | 5 990 ₽ | 4 модуля: метрики + фреймворк кейсов + кейсы по компаниям + практикум. Видеоуроки + текстовые материалы + Excel-практика |
| Продвинутый | 12 990 ₽ | Всё из базового + 4 записи реальных собеседований с разбором + кейсы-блоки для собесов |
| С консультацией | 19 990 ₽ | Всё из продвинутого + 1 личная консультация с Алексеем (лимит 4 места/мес) |

**Стадия:** Модуль 1 готов (27 уроков). Модули 2–4 в разработке. Запуск: апрель 2026.

**Содержание курса:**
- Модуль 1: Продуктовые метрики (DAU/MAU, retention, воронки, монетизация, unit-экономика, когорты, статистические ловушки)
- Модуль 2: Фреймворк решения кейсов (КМДГП: Контекст → Метрика → Декомпозиция → Гипотезы → Проверка)
- Модуль 3: Кейсы по компаниям (маркетплейсы, финтех, поиск и рекомендации)
- Модуль 4: Практикум (самостоятельные кейсы без подсказок)

### 3.2 Миникурс: АБ-тесты

**Позиционирование:** «20/80: 20% знаний, с помощью которых ты пройдёшь 80% собесов по АБ-тестам.»

**ЦА:** аналитики перед собеседованиями, нет опыта проведения АБ.

| Тариф | Цена | Состав |
|-------|------|--------|
| Теория | 5 990 ₽ | Статистика для АБ, чек-лист из 16 пунктов, дизайн АБ-теста (шаги), где взять опыт без реального АБ, записи собеседований с вопросами |
| Теория + практика | 12 990 ₽ | Всё из теории + ручной анализ 3 АБ-тестов + работа с критериями, MDE, ratio-метриками |
| Теория + практика + 2 консультации | 24 990 ₽ | Всё выше + 2 консультации с Алексеем |

**Стадия:** есть MVP, продавался несколько раз (связка консультация 6К + курс 6К). Требуется переделка для внешней продажи.

**Планы:** имитация платформы АБ-тестов (по аналогии с GrowthBook) — ученик заносит метрики, считает длительность. 5 симулированных АБ-тестов для расширенных тарифов.

### 3.4 Road Map «Фундамент аналитика»

**Позиционирование:** «Всё что нужно, чтобы получить оффер на 200К и вырасти до 400К. Материалы для подготовки и прохождения собесов, составленные на личном опыте и опыте нескольких десятков учеников.»

| Параметр | Значение |
|----------|----------|
| Цена | 19 900 ₽ |
| Состав | SQL + Python/Pandas + Продуктовые кейсы и метрики + АБ-тесты и статистика + Тервер + Резюме |
| Скидка vs отдельные курсы | ~40–45% (сумма отдельных курсов ~30–36К) |

**Стадия:** зависит от разработки отдельных миникурсов. По сути бандл из миникурсов.

**Upsell-механика:** на лендинге каждого миникурса показывать «Или купи всё сразу за 19 900 вместо 35 940». При покупке миникурса — разница засчитывается при апгрейде до Road Map в течение 30 дней.

## 4. Ценовая архитектура и upsell-цепочки

### Воронка продуктов

```
Бесплатный контент (YouTube / TG / Shorts / Reels / TikTok / VK)
    ↓
Лид-магнит: «Чек-лист 16 продуктовых метрик для собеседования» → TG-бот / закрытый канал
    ↓
Миникурс 5 990 ₽ (продуктовая аналитика или АБ-тесты)
    ↓
Расширенный тариф 12 990 ₽ (практика + записи собесов)
    ↓
Тариф с консультацией 19 990–24 990 ₽
    ↓
Road Map бандл 19 900 ₽ («возьми всё со скидкой»)
    ↓
Bootcamp 29 900 ₽ («получи реальный опыт»)
    ↓
[Для единиц] Менторство до оффера
```

### Upsell-точки

| Момент | Действие |
|--------|----------|
| После покупки миникурса | Через 3 дня в TG-боте: «Road Map со скидкой — разница в цене засчитывается» |
| После прохождения Road Map | Предложение Bootcamp |
| Тариф с консультацией | Естественная точка для упоминания менторства (не навязывать) |
| Покупка одного миникурса | На странице благодарности: «Возьми второй курс со скидкой 20%» |

### Ограничение тарифа с консультацией

- Лимит: 4 места в месяц
- На лендинге показывать «осталось X мест» (реальное ограничение, не маркетинговая уловка)
- Не делегировать консультации — пока это личная экспертиза Алексея

**Короткие видео (все площадки):**

- Формат: 1 видео = 1 метрика / 1 совет / 1 ошибка

## 11. Бета-тест перед запуском

### Формат
Дать 3–5 текущим ученикам менторства доступ к курсу на 5 дней.

### Что спрашивать
1. Пройди от начала до конца, отметь где застрял / непонятно / скучно
2. Какой урок самый полезный?
3. Чего не хватает? (конкретно — пример, задача, объяснение)
4. Запиши голосовое 2 минуты — что думаешь после прохождения
5. Кому из знакомых аналитиков ты бы скинул этот курс?

### Что НЕ спрашивать
- «Купил бы?» — бесполезный вопрос, ответ всегда «да» из вежливости
- «Сколько бы заплатил?» — ученики получили бесплатно, контекст другой

### Использование результатов
- Цитаты из голосовых → отзывы на лендинг
- «Самый полезный урок» → акцент в рекламе
- «Чего не хватает» → доработка перед запуском
