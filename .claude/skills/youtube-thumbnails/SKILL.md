# Skill: Создание обложек и заголовков YouTube с высоким CTR

## Когда использовать

При создании или улучшении обложек (thumbnails) и заголовков для YouTube-канала «Аналитика без воды». Вызов: `/youtube-thumbnails`

## Контекст канала

- **Канал:** Аналитика без воды (@analytic_offers)
- **Ниша:** карьера в аналитике, переход в профессию, подготовка к собеседованиям
- **Аудитория:** начинающие аналитики, переходящие из других профессий, 25-35 лет
- **Средний CTR канала:** 5.4% (за всё время, 977K показов)
- **Формат:** лонги (11 видео) + shorts (23 видео)

### Текст на обложке — главный рычаг

| Слов | Эффект | Источник |
|---|---|---|
| 0–3 | Оптимум, максимальный CTR | A/B тесты (Ali Abdaal, Malewicz) |
| До 6 | +24% vs базы | TubeBuddy study |
| 10+ | **-16% к CTR** | TubeBuddy study |

**A/B кейс:** 12 слов на тёмном фоне → 3 слова на жёлтом фоне = CTR с 2% до 7% (3.5x). Источник: BananaThumbnail.

### Цвета

| Комбинация | CTR / эффект | Когда |
|---|---|---|
| Синий + оранжевый | +25% CTR | **Tech/Education — наш выбор** |
| Жёлтый + чёрный | 8.2% avg (лучшая) | Предупреждения, туториалы |
| Красный + белый | +30% vs нейтральных | Провокации, ошибки |

### Лица и эмоции

| Фактор | Данные |
|---|---|
| Лицо с эмоцией vs без лица | 9.2% vs 6.1% CTR (TubeBuddy, 1.2M видео) |
| Удивление (surprise) | +49% подъём CTR |
| «Настоящая» эмоция vs «YouTube face» | 73% зрителей предпочитают настоящую |
| Закрытый рот vs открытый | Закрытый = лучше watch time (MrBeast) |
| «Лица всегда работают» | **МИФ** — зависит от ниши (SearchEngineJournal, 300K видео) |

### Контраст и мобильная версия

| Фактор | Эффект |
|---|---|
| Контраст > 4.5:1 | +31% CTR на мобильных |
| Низкий контраст | -22% алгоритмическое понижение |
| Тёмный фон в Dark Mode YouTube | -40-60% потенциального CTR |
| 72.1% просмотров — мобильные | Если не читается на 320x180 — не работает |

### Сложность обложки

**A/B кейс:** Malewicz убрал загруженный фон → один цвет + «Don't do it!» = CTR с 5% до 10.2% (2x). Использовал AI eye-tracking для диагностики split-focus.

**Правило:** один фокальный элемент > несколько конкурирующих.

### Консистентность бренда

| Стиль обложек | Click-to-Subscribe |
|---|---|
| Консистентный | 12.9% |
| Непоследовательный | 4.0% |

### A/B тестирование

- YouTube Studio Test & Compare даёт в среднем **+20% CTR** даже без знаний дизайна
- MrBeast: ~50 концептов → 12 A/B вариантов через TubeBuddy на каждое видео
- Медианный подъём после систематического тестирования: **+33%** (с 4.1% до 5.0-5.5%)

## Стандарты YouTube-обложек

### Технические требования

- **Разрешение:** 1280 x 720 px (минимум), рекомендуется 1920 x 1080
- **Соотношение сторон:** 16:9
- **Формат:** JPG, PNG, GIF (без анимации)
- **Размер:** до 2 МБ
- **Safe zone:** основной контент в центральных 80% (края обрезаются на мобильных)

### Правило мобильного теста

> 70% просмотров YouTube — с мобильных. Обложка отображается размером ~3.5 x 2 см.
> Если текст или элемент не виден на экране телефона — он бесполезен.

Проверка: уменьши обложку до 320 x 180 px. Если текст нечитаем — переделывай.

### Зоны внимания (heat map)

```
┌─────────────────────────────┐
│  ████████           ██████  │  ← верхняя треть: заголовок
│  ████████           ██████  │
│                             │
│       ████████████          │  ← центр: лицо / главный объект
│       ████████████          │
│                             │
│  ██████           ████████  │  ← нижняя треть: цифры / CTA
│  ██████           ████████  │
└─────────────────────────────┘
     ↑ YouTube ставит          ↑ Длительность видео
       значок канала             (правый нижний угол)
```

**Важно:** правый нижний угол перекрывается таймкодом длительности — не ставить туда текст.

## Формулы обложек с высоким CTR

### Формула 1: Split (до/после)

```
┌──────────────┬──────────────┐
│   ГРУСТНОЕ   │  СЧАСТЛИВОЕ  │
│    ЛИЦО      │    ЛИЦО      │
│              │              │
│   150 000 ₽  │   500 000 ₽  │
│   [Excel]    │   [Python]   │
└──────────────┴──────────────┘
```

- Работает для: сравнений, трансформаций, выбора
- CTR на канале: 5.5-5.6%
- Правила: левая часть = боль (тёмная), правая = результат (светлая)

### Формула 2: Провокация / вопрос

```
┌─────────────────────────────┐
│                             │
│   [ОБЪЕКТ]    [ЛИЦО с       │
│   разбитый/   эмоцией]      │
│   перечёркн.                │
│                             │
│   СЛОВО МЕРТВО?             │
│   [год]                     │
└─────────────────────────────┘
```

- Работает для: трендовых тем, разрушения мифов
- CTR на канале: 6.8%
- Правила: 2-4 слова макс, вопросительный знак, конкретный год

### Формула 3: Бренд + инсайд

```
┌─────────────────────────────┐
│                  [ЛОГОТИП]  │
│   [ЛИЦО]        крупно     │
│   серьёзное                 │
│                             │
│   БРЕНД ИЗНУТРИ             │
│   Senior Analyst            │
└─────────────────────────────┘
```

- Работает для: кейсов из компаний, реального опыта
- CTR на канале: 7.6% (лучший результат)
- Правила: узнаваемый логотип, авторитетная поза, минимум текста

### Формула 4: Конкретный результат

```
┌─────────────────────────────┐
│                             │
│   [ЛИЦО]     [ЭКРАН с      │
│   показывает  конкретным    │
│   на экран]   результатом]  │
│                             │
│   ИНСТРУМЕНТ + РЕЗУЛЬТАТ    │
└─────────────────────────────┘
```

- Работает для: инструментов, AI, практических тем
- CTR на канале: 5.9%
- Правила: на экране — читаемый код/результат, не каша

### Формула 5: Числовой список

```
┌─────────────────────────────┐
│         ТОП-N               │
│                             │
│   [ЛИЦО]   [2-3 крупных    │
│             логотипа]       │
│                             │
│   ТЕМА СПИСКА               │
└─────────────────────────────┘
```

- Работает для: подборок, рейтингов
- CTR на канале: 4.7-4.8%
- Правила: максимум 3-4 логотипа, не больше. Число крупно.

### Формула 6: Скриншот-провокация (NEW, подтверждено VPD)

```
┌─────────────────────────────┐
│  ПРОВОКАЦИЯ!       [ЛИЦО    │
│                     с WTF-  │
│  ┌──────────────┐  реакцией]│
│  │ Скриншот     │           │
│  │ hh.ru /      │           │
│  │ LinkedIn /   │           │
│  │ вакансии     │           │
│  └──────────────┘           │
└─────────────────────────────┘
```

- Работает для: разборов рынка, зарплат, абсурдных вакансий
- VPD референсы: Theo 24K/day, Joshua Fluke 16K/day
- Правила: реальный скриншот (50-60%), лицо с реакцией сбоку, 2-3 слова сверху
- **Для канала:** скриншот hh.ru + лицо + «300К ОФФЕР» или «15К ЗАРПЛАТА?!»

### Формула 7: Минимал-драма (NEW, подтверждено A/B)

```
┌─────────────────────────────┐
│                             │
│   [ЛИЦО              ВСЁ.  │
│    крупным            или   │
│    планом,           ОФФЕР  │
│    задумчивое]        300К  │
│                             │
└─────────────────────────────┘
```

- Работает для: кейсов учеников, карьерных поворотов, итогов
- A/B данные: Malewicz 5% → 10.2% через упрощение до одного фокуса
- Правила: лицо на 50% обложки + 1-2 слова ГИГАНТСКИМ шрифтом + простой фон
- **Для канала:** кейс ученика «ОФФЕР 350К», карьерные «IT — ВСЁ?»

## Правила заголовков (Title)

### Структура заголовка

```
[Хук/интрига] + [Конкретика] + [Для кого / контекст]
```

**Примеры хороших заголовков канала:**
- «Excel vs Python: почему одни аналитики застряли на 150 000, а другие вышли на 500 000»
- «Эра дашборд-аналитиков закончилась. Что делать в 2026?»
- «Что РЕАЛЬНО делает аналитик в Wildberries (не дашборды)»

### Что работает в заголовках

| Приём | Пример | Зачем |
|---|---|---|
| Конкретные цифры | «150 000 vs 500 000» | Количественная интрига |
| Год / актуальность | «в 2026» | Свежесть контента |
| Скобки с уточнением | «(не дашборды)» | Ломает ожидания |
| CAPS на 1-2 словах | «РЕАЛЬНО», «ПРАВИЛЬНО» | Акцент без крика |
| Вопрос или утверждение | «Что делать?» | Вовлечение |
| Бренд / компания | «в Wildberries» | Узнаваемость + конкретика |

### Чего избегать в заголовках

- Больше 60 символов (обрезается на мобильных)
- ВСЕ СЛОВА КАПСОМ (воспринимается как спам)
- Кликбейт без покрытия в видео (убивает retention и доверие)
- Общие фразы без конкретики («Всё о профессии», «Полный гайд»)
- Эмодзи в заголовке (допустимо 1 максимум, лучше без)

### Формула: заголовок + обложка = одно сообщение

> Заголовок и обложка НЕ должны дублировать друг друга.
> Обложка — визуальный хук (эмоция, бренд, цифры).
> Заголовок — текстовый контекст (о чём, для кого, почему сейчас).

**Плохо:** обложка «ТОП-6 ИНСТРУМЕНТОВ» + заголовок «ТОП-6 нужных инструментов аналитика» → дубль.

**Хорошо:** обложка с логотипами SQL/Python/Pandas + заголовок «Вот что на самом деле используют аналитики: ТОП-6 нужных инструментов» → обложка показывает ЧТО, заголовок добавляет КОНТЕКСТ.

## Чек-лист перед публикацией

### Обложка

- [ ] **Максимум 3 слова** текста (доказано: 3 слова = оптимум, 10+ = -16% CTR)
- [ ] Читается на мобильном (320x180 px тест, 72% аудитории)
- [ ] Контраст > 4.5:1 (иначе -31% CTR на мобильных, -22% от алгоритма)
- [ ] **Нет тёмного фона** (Dark Mode YouTube = потеря 40-60% CTR)
- [ ] Лицо с настоящей эмоцией (не «YouTube face» — 73% предпочитают настоящую)
- [ ] Один фокальный элемент (Malewicz: упрощение = 2x CTR)
- [ ] 2-3 крупных элемента, не больше
- [ ] Нет текста в правом нижнем углу (перекроется таймкодом)
- [ ] Не дублирует текст заголовка
- [ ] Соответствует одной из формул ниже
- [ ] Цвета: синий + оранжевый (tech/edu: +25%) или жёлтый + чёрный (8.2% avg)

### Заголовок

- [ ] До 60 символов (проверить обрезку)
- [ ] Есть конкретика (цифра, бренд, год)
- [ ] CAPS максимум на 1-2 словах
- [ ] Дополняет обложку, а не дублирует
- [ ] Первые 40 символов — самые важные (видны везде)

## Workflow создания обложки

1. **Определи формулу** — какая из 7 формул подходит под тему видео
2. **Напиши текст обложки** — максимум 2-3 слова (доказано: оптимум)
3. **Выбери цвета** — синий+оранжевый (tech/edu) или жёлтый+чёрный (туториалы)
4. **Выбери эмоцию** — настоящая, не «YouTube face» (73% предпочитают)
5. **Один фокус** — убери всё лишнее, один главный элемент (Malewicz: 2x CTR)
6. **Мобильный тест** — уменьши до 320x180, проверь контраст > 4.5:1
7. **Проверь пару** — заголовок + обложка = одно сообщение без дублей
8. **Пройди чек-лист** — все пункты выше
9. **Запусти A/B тест** — YouTube Studio Test & Compare (avg +20% CTR)

## Workflow улучшения существующей обложки

1. Проверь CTR видео в YouTube Studio (Аналитика → Расширенный режим → Показы + CTR)
2. Если CTR < 4% для лонга — обложку нужно менять
3. Сравни с лучшими обложками канала (7.6%, 6.8%, 5.9%)
4. Определи проблему по таблице «что убивает CTR»
5. Переделай по подходящей формуле
6. A/B тест: YouTube позволяет тестировать обложки (Thumbnail Test & Compare)

## Приоритет переделки обложек

По данным CTR-аудита — обложки с CTR < 4.5%, которые нужно переделать первыми:

| CTR | Видео | Проблема | Решение (формула) |
|---|---|---|---|
| **2.8%** | 4 кейса WB | Абстрактные иконки + кликбейт «МИЛЛИОНЫ» | Формула 3 (бренд WB крупно + лицо) или Формула 7 (минимал: «WB КЕЙСЫ» + лицо) |
| **3.2%** | IT профессии | 7 мелких логотипов + длинный текст | Формула 7 (минимал-драма: лицо + «КУДА В IT?» 2 слова) |
| **4.2%** | ТОП-10 вопросов | Нечитаемый текст на ноутбуке | Формула 6 (скриншот вопроса + лицо с реакцией) или Формула 7 |
