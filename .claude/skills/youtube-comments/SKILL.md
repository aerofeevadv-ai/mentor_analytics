## Workflow

```
1. Claude парсит комменты без ответа → заливает в Google Sheets
2. Claude классифицирует + пишет черновики ответов → обновляет таблицу
3. Алексей проходит по таблице:
   - Если ответ ок → копирует и публикует на YouTube вручную
   - Если нужна правка → пишет комментарий в «Правка»
   - Если игнор → ничего не делает
4. Если Алексей написал правки → Claude переписывает черновики
5. Claude записывает дату обработки в comments_log.md
```

**ВАЖНО:** Claude НЕ публикует ответы на YouTube. Алексей делает это вручную.

## Шаг 2: Классификация

Прочитай `.secrets/fresh_comments.json` и классифицируй каждый комментарий.

### УДАЛИТЬ (Действие = «удалить»)

Удаляй без вопросов:
- Мат, оскорбления
- Тролли: провокации, «одна вода», «бред», «зачем это видео»
- Обвинения в использовании нейросетей: «GPT писал», «через нейронку», «бот отвечает»
- Спам, реклама чужих каналов/курсов
- Политические провокации («на фронт пойдёшь» и подобное)

### ИГНОРИРОВАТЬ (Действие = «игнор»)

- Односложные: «Спасибо», «Круто», «Топ» без лайков — просто лайкнуть
- Эмодзи-только комментарии
- Комментарии не про автора/канал (обсуждают WB, рынок, политику между собой)
- Комментарии старше 60 дней без лайков

### ОТВЕТИТЬ (Действие = «ответить»)

Всё остальное. Приоритет ответов:

1. **Лиды** — любой вопрос про менторство, обучение, «берёте учеников?», «как записаться?»
2. **Вопросы с лайками (5+)** — высокая видимость, работает на имидж
3. **Повторные зрители** — комментят под несколькими видео
4. **Свежие видео без ответов** — видео за последние 30 дней с 0 ответами автора
5. **Остальные вопросы** — технические, карьерные, возрастные

## Шаг 4: Обновить Google Sheets

Прочитай ID таблицы из `.secrets/comments_sheet_id.txt`.

Обнови колонки в таблице через Google Sheets API (bash + python):

```python
# Для каждого комментария обновить:
# - Тональность (колонка E)
# - Действие (колонка F): ответить / удалить / игнор
# - Предложенный ответ (колонка G): черновик ответа
```

Группировать строки по видео (все комменты к одному видео вместе).

Колонки таблицы:
| Видео | Автор | Комментарий | Лайки | Тональность | Действие | Предложенный ответ | Правка | comment_id |

- **Действие**: dropdown (ответить / удалить / игнор)
- **Правка**: пустая колонка — Алексей пишет сюда если хочет изменить ответ
- **comment_id**: скрытая колонка

После обновления скажи Алексею:
```
Таблица готова: https://docs.google.com/spreadsheets/d/{SHEET_ID}

- К ответу: XX
- К удалению: XX
- К игнору: XX

Последний раз комментарии обрабатывались: {ДАТА}

Скопируй ответы и опубликуй на YouTube.
Если хочешь изменить ответ — впиши текст в «Правка», я перепишу.
```

## Стиль ответов (СТРОГО)

### Делай
- Пиши коротко, 1-3 предложения
- Разговорный живой язык, как в чате
- Высказывай своё мнение
- Конкретные советы вместо общих слов
- Если есть релевантное видео на канале — давай ссылку
- На вопрос про менторство: «Подробнее про менторство тут https://mentoranalytics.ru/personal-mentor»

### Не делай
- Не используй длинные тире (—), только короткие (-)
- Не используй нумерованные списки (1. 2. 3.)
- Не используй буллеты
- Не начинай с «Отличный вопрос!», «Спасибо за комментарий!»
- Не используй канцелярит: «следует отметить», «хотелось бы подчеркнуть», «данный»
- Не используй эмодзи в тексте (кроме ссылок)
- Не пиши «как я уже говорил в видео» — зритель мог не досмотреть
- Не упоминай AI/нейросети в ответах (не кормить троллей)

### Примеры хороших ответов

На вопрос про возраст:
> 30 - отличное время. У меня куча менти этого возраста, многие уже с офферами. Главное не растягивать обучение на годы, а за 4-6 месяцев дойти до первых проектов и собесов

На техвопрос:
> Не нужен «мидл Python». Нужно: pandas, numpy, matplotlib/seaborn, базовые конструкции. Это 2-3 месяца, если учить параллельно с SQL

На скепсис про AI:
> Сократят тех, кто делает одно и то же руками. Аналитики, которые используют AI как инструмент - наоборот, становятся дороже
