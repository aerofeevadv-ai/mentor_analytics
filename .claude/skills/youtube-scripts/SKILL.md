# YouTube Scripts — Полный продакшн-пакет видео

Skill для создания видео от идеи до готового сценария. Включает: анализ референсов, тезисный план, полный сценарий для телесуфлёра, ТЗ монтажёру, ТЗ на обложку, SEO-блок, план shorts, retention-ревью.

## Контекст канала

- **Канал:** Аналитика без воды (@analytic_offers)
- **Ведущий:** Лёха, Senior Product Analyst в Wildberries (ex-Яндекс)
- **Аудитория:** аналитики 25-35 лет, которые хотят больше зарабатывать
- **Позиционирование:** практика без воды, фокус на доход и офферы
- **Тон:** прямой, экспертный, разговорный, без академизма
- **Рынок:** РФ и СНГ (hh.ru, российские компании, зарплаты в рублях)
- **Менторство:** boutique «до оффера», не онлайн-школа
- **Формат:** talking head + скринкасты, 8-15 минут

## Шаг 4: Полный сценарий для телесуфлёра

Создать файл `content/youtube/SLUG/script.md`.

### Retention-маркеры в тексте

Вставлять в сценарий маркеры — для автора при записи и монтажёра:

```
🔴 OPEN LOOP: [тизер — «Через минуту покажу ошибку, которая стоит 100К»]
🟡 СМЕНА ЭНЕРГИИ: [что меняется — вопрос, история, скринкаст, юмор]
🟢 ГРАФИКА: [что показать — текст на экране, схема, скриншот]
📱 СКРИНКАСТ: [что показать на экране]
✂️ SHORTS: [этот фрагмент можно вырезать в short]
```

### Формат файла

```markdown
# Сценарий: [Название]

**Хронометраж:** ~X минут
**Тип:** обучающее / карьерное / кейс
**CTA:** [что продаём / куда ведём]
**Обложка:** [рекомендуемая формула из youtube-thumbnails]
**Бриф:** brief.md

---

[00:00] **ХУК**

🔴 OPEN LOOP: [тизер — зачем досматривать]

[Текст хука. Короткие предложения.]

---

[00:30] **Блок 1: [Название]**

🟡 СМЕНА ЭНЕРГИИ: переход от хука к контенту

[Текст блока.]

🟢 ГРАФИКА: [что показать на экране]

[Продолжение текста.]

✂️ SHORTS: фрагмент 00:45-01:15

---

[03:00] **Блок 2: [Название]**

🔴 OPEN LOOP: [тизер следующего блока]

📱 СКРИНКАСТ: [что показываем]

[Текст.]

---

[XX:XX] **ЗАКРЫТИЕ**

[Резюме 2-3 тезиса + CTA]

---

## Статус
DRAFT
```

### Правила текста

**Стиль:**
- Короткие предложения — максимум 15-20 слов. Одна мысль = одно предложение
- Обращение на «ты» — всегда
- Разговорный, как объясняешь другу. Без «следует отметить», «данный»
- Конкретика: числа, примеры, названия компаний
- Своё мнение: «Я считаю», «По моему опыту», не «Эксперты говорят»
- Паузы через «...» — для интонационных акцентов
- Вопросы к зрителю: «Окей, логичный вопрос: а что учить?»
- Личные истории из первого лица: «Когда я работал в Яндексе...»
- Зарплаты в рублях, компании российские

**Эталон тона (из видео канала):**
- «Я сам через это прошёл. Учил нампай, проходил бесполезные курсы»
- «Это как разукрашивать фасад дома, если нет фундамента»
- «Поздравляю, ты на коне»
- «Я не гений, не математик, не олимпиадник. Я просто прошёл этот путь»

**Запрещено:**
- «Привет, друзья! С вами снова...» — начинать с хука
- Канцелярит: «хотелось бы отметить», «существует множество подходов»
- Перечисления длиннее 5 пунктов подряд
- Абстрактные обещания: «Всё станет ясно»
- Ремарки в тексте суфлёра ([пауза], (смех)). Заметки — в отдельном блоке

**Ритм:**
- Каждые 60-90 секунд — смена энергии (🟡 маркер)
- Каждые 2-3 минуты — open loop (🔴 маркер)
- Нет монолога длиннее 90 секунд без перебивки

## Шаг 5: ТЗ монтажёру

Создать файл `content/youtube/SLUG/editing_brief.md`:

```markdown
# ТЗ монтажёру: [Название видео]

## Общий стиль
- Темп: [быстрый / средний / спокойный]
- Настроение: [энергичный / серьёзный / ироничный]
- Референс монтажа: [ссылка на видео с похожим стилем]

## Покадровый план

| Таймкод | Что на экране | Тип | Заметки |
|---|---|---|---|
| 00:00-00:05 | Talking head крупный план | Hook | Сразу с первого слова, без заставки |
| 00:05-00:15 | Текст на экране: «[ЦИТАТА]» | Lower third | Крупный шрифт, 2-3 слова |
| 00:15-00:30 | Talking head + zoom in | Акцент | Zoom на ключевой фразе |
| ... | ... | ... | ... |

## Правила монтажа для retention

### Первые 30 секунд (критично)
- БЕЗ заставки, лого, интро
- Начинать с mid-sentence или провокации
- Первый cut через 3-5 секунд (убрать паузы)
- Текст на экране в первые 5 секунд

### Ритм кадра
- Менять ракурс/план каждые 15-30 секунд
- На длинных монологах (>20 сек) — jump cut или zoom
- Каждый новый тезис = новый кадр

### Текст на экране
- Ключевые цифры и тезисы — показывать текстом
- Читаемый на мобильном (крупный, контрастный)
- Не больше 5 слов на экране одновременно

### Скринкасты
- Подсветка курсора / области
- Zoom на важных элементах
- Не показывать пустой экран > 3 секунд

### Звук
- Фоновая музыка тихо, под голос
- Звуковой акцент (whoosh/click) на переходах между блоками
- Убрать все паузы > 0.5 секунды

| 2 | XX:XX-XX:XX | [тема] | [первая фраза] | ~55 сек |
```

## Шаг 7: Retention-ревью (самопроверка)

После написания сценария — провести полный аудит.

### 7a. Поминутная retention-карта

```
[ММ:СС–ММ:СС] Retention: ↑/→/↓ | Что происходит | Риск/сила

[00:00–00:25] ↑↑ | Провокация + credential | Сильный хук
[00:25–01:00] →  | Контекст | Нормально, нет личной истории
[01:00–01:30] ↓  | Теория без примера | ⚠️ ПРОВАЛ — добавить кейс
[01:30–02:00] ↑  | Личная история | Эмоциональный пик
```

### 7b. Скоринг по 7 критериям

| Критерий | Вес | Шкала | Что оцениваем |
|---|---|---|---|
| **Хук** | 25% | 1-10 | Сила первых 10 секунд |
| **Плотность хуков** | 20% | 1-10 | Open loops, истории, цифры каждые 30-60 сек |
| **Монотонность** | 15% | 1-10 | Нет участков >90 сек без смены формата (10 = хорошо) |
| **Личные истории** | 10% | 1-10 | Минимум 2 за видео |
| **Конкретика** | 10% | 1-10 | Цифры, компании, зарплаты vs абстракции |
| **Финал** | 10% | 1-10 | Сильное закрытие + CTA |
| **Читаемость вслух** | 10% | 1-10 | Предложения ≤20 слов, разговорный |

| Итог | Вердикт |
|---|---|
| 8.5-10 | Публиковать. Минимальные правки |
| 7.0-8.4 | Хорошо, но есть точки усиления |
| 5.0-6.9 | Переработать конкретные блоки |
| < 5.0 | Переписать |

Для каждой проблемы — конкретная правка:

```
❌ БЫЛО:
«Существует множество подходов к изучению аналитики»

✅ СТАЛО:
«Есть 3 рабочих способа войти в аналитику. Первый — самый быстрый, но с нюансом...»

📊 Почему: абстракция → конкретика + open loop. Retention ↓ → ↑
```

```markdown

- [ ] 🟡 смена энергии каждые 60-90 секунд
- [ ] 🔴 open loop каждые 2-3 минуты
- [ ] Нет монолога > 90 секунд без перебивки
- [ ] Нет перечислений > 5 пунктов подряд

### Контент
- [ ] Каждый блок — конкретный пример или кейс
- [ ] Своё мнение / позиция
- [ ] Нет «воды» — абзацев, которые можно удалить без потери

- [ ] Хронометраж 8-15 минут
- [ ] Заголовок + обложка не дублируют друг друга
- [ ] Минимум 2 shorts-фрагмента отмечены ✂️
- [ ] Все примеры — российские (компании, зарплаты, платформы)

### Прогноз
**Retention 30 сек:** X% (цель: >70%)
**Средний retention:** X% (цель: >45%)
```

## Структурные паттерны видео

### Паттерн A: Ошибки → Решение → Roadmap (10-20 мин)
```
Хук → Credential → Личная история →
Ошибка 1 → 2 → 3 → 4 →
«Ок, тогда что реально учить?» →
Блок 1 → 2 → 3 → 4 →
Roadmap → Практика → Упаковка →
CTA
```
Пример: «Как ПРАВИЛЬНО изучать аналитику» — 40K views.

### Паттерн B: Провокация → 3 тренда → CTA (5-7 мин)
```
Провокационный хук → Credential →
Контекст проблемы →
Тренд 1 (+пример) → 2 → 3 →
Резюме: что делать →
CTA
```
Пример: «Эра дашборд-аналитиков» — 6.8% CTR.

### Паттерн C: Кейс/разбор (8-12 мин)
```
Хук с результатом → Credential →
Контекст: кто, откуда →
Проблема → Решение шаг 1 → 2 → 3 →
Результат с цифрами →
Выводы → CTA
```

### Паттерн D: Адаптация конкурента
```
Взять структуру оригинала →
Заменить примеры на российские →
Добавить личные истории →
Усилить хук →
Переписать в тоне канала →
Добавить retention-маркеры →
Адаптировать CTA
```

## Retention-механики (каждые 30-60 секунд)

| Механика | Пример | Зачем |
|---|---|---|
| **Open loop** | «Дальше покажу, что реально спрашивают на собесах» | Зритель ждёт |
| **Перечисление** | «4 ошибки, которые сжирают время» | Считает: была 1-я, жду 2-ю |
| **Pattern interrupt** | «Стоп. Давай с другой стороны» | Сброс монотонности |
| **Personal story** | «Помню свой первый собес — руки тряслись» | Эмоциональная привязка |
| **Конкретные цифры** | «150 000 vs 500 000» | Якорь внимания |
| **Milestone reward** | «Если ты досмотрел до сюда, у тебя уже есть вопрос...» | Награда за досмотр |
| **Переход-мост** | «Окей, с ошибками разобрались. Теперь главный вопрос.» | Плавный переход |

## Retention-убийцы (чего избегать)

| Убийца | Что делать вместо |
|---|---|
| Длинное вступление (>30 сек) | Хук → credential → обещание за 20 сек |
| Монотонные перечисления | После каждого пункта — история или пример |
